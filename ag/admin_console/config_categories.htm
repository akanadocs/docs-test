Configuration Categories and Settings Guide

This guide provides a brief explanation of the various configuration categories and settings in Policy Manager. In many cases, the solution to a specific issue is to modify a configuration setting. This section describes the settings that are available.

SOA Software Administration Console Configuration Categories and Properties

The sm70\docs\configAdmin folder of the Policy Manager 7.2 release directory now includes documentation for Configuration Categories that are available via the Configuration tab of the SOA Software Administration Console. The index.htm file presents a list of Configuration Categories and list of supported properties with a brief description of the property and default state (if applicable).
