Introduction

This feature includes a web based user interface for the central Policy Manager governance application. The user interface provides a dashboard and the primary interface for configuration, management and administration of Policy Manager and Service Manager. The console must interact with a Container (may be the same container) configured with the SOA Software Policy Manager Services to fulfill the core central governance application requirements.

Step 1: Installation Prerequisites

  • SOA Software Platform, Version 6.x or 7.x must be installed.
  • A Policy Manager container must be defined using the Configure Container Instance Wizard.

Step 2: Install SOA Software Policy Manager Console Feature

  1. Launch the SOA Software Administration Console (http://<hostname:<port>/admin/).
  2. On the Installed Features screen, select Feature from the Filter menu.
  3. Select the SOA Software Policy Manager Console feature and click Install Feature.
  4. After the installation is complete, click Configure and continue to Step 3.

Step 3: Define Policy Manager Administrator Credentials

  1. On the Define Policy Manager Administrator Credentials screen, specify login credentials for the Policy Manager Management Console, or accept the default (administrator/password).

Step 4: Log Into the Policy Manager Management Console

  1. Log into the Policy Manager Management Console (http://<hostname:<port>/ms/index.do/) and begin your service management tasks.
