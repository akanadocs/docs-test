Caching information (com.soa.http.clientcaching)

Used for creation of Response HTTP headers that are sent to browsers and used for caching settings

Client Caching (clientCachingEnabled)

Turn client caching on or off for static resources

Default: true

Client Cache Time (clientCacheTime)

Set the length of time for client to cache static resources. This is reflected in the Expires and Cache-Control headers Default setting is 1 year

Default: 31536000
