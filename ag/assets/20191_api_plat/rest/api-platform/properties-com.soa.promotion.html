Promotion (com.soa.promotion)

Promotion configuration properties

Promotion Coordinator Address (remote.promotor.address)

The address (URL) of the remote promotion coordinator (if any)

Maximum Async Executor Threads (tenant.api.maxAsyncExecutorThreads)

The maximum threads allowed to asynchronously process promotion requests

Default: 8

Maximum Async Promotion Cache Minutes (tenant.api.maxAsyncPromotionCacheMinutes)

The maximum minutes allowed to asynchronously process promotion requests

Default: 60
