Api Security (com.soa.api.security)

Api security cache configuration properties

Refresh Time (com.soa.api.security.cache.refreshTime)

The refresh time in milliseconds the cache will be refreshed.

Default: 300000

Expiration Period (com.soa.api.security.cache.expirationPeriod)

The expiration period in milliseconds the cache will be expired.

Default: 3600000
