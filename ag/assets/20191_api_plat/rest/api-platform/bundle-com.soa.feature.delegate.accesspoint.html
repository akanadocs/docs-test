Akana Delegate Access Point com.soa.feature.delegate.accesspoint/2019.1.0

This feature provides a remote interface to delegates for policy implementation and message delivery. Any delegate that does not provide its own policy implementation and message delivery mechanism can route messages through the Access Point. This feature is installed on a container that does not have native delegate support.
