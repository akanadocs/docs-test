Http Security Policy (com.soa.policy.handler.httpsecurity)

Configuration of the Http Security policy handler

Keep Security Headers (http.security.handler.factory.keepHeaders)

Indicates whether or not to keep HTTP security headers after they've been successfully processed. This setting is on the policy handler factory so changes won't be reflected until new policies are constructed or reattached. It is recommended to restart the container to see consistent results.

Default: true
