Rhino Javascript Engine (com.soa.rhino.scriptengine)

Configuration for the Rhino Javascript engine

Java Class Instantiation Restriction (rhino.java.class.shutter.restricted)

Restrict java class instantiation. When unrestricted any Java class loaed in the current process can be instantiated from Javascript. To restrict to only specified packages and/or classes set this property to true.

Default: false

Allowed Java Classes (rhino.java.class.shutter.classesAndPackages)

List of java packages and/or classes that can be instantiated from Javascript in one delimited string. The format is full package names or full class names separated by a ''.

Default: java.iojava.langjava.mathjava.text
