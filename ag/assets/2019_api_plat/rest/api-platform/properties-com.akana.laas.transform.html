Lifecycle Manager Asset Transformation (com.akana.laas.transform)

Lifecycle Manager Asset Transformation Service Configuration

Default Format (default)

The default format for transformations if none specified.

Default: alpaca

Alpaca Template (alpaca)

The template for Alpaca tranformation. Other templates may be specified in the same way, using format=template name. The template name can be a file name or the name of a resource, which by default is a built-in internal template.

Default: templates/asset.alpaca.vm
