Akana Scheduled Jobs com.soa.feature.jobs.engine/2019.0.0

Installs a series of system background jobs that are executed by the Policy Manager job scheduler. This feature is used for multi-container deployments and is also installed as part of the Akana Policy Manager Services feature (for single container deployments) along with Akana Managed Services, and Akana Security Services features.
