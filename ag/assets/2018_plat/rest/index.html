Public APIs for 2018.0.9

[Endpoints]
Bundle Administration Service
API used to manage system bundles.
Configuration Admin API
A REST API that provides access to the Configuration Admin service.
Feature Administration Service
This API is used to administer the installation,removal, and resolution of features.
Health Service

The Health Service provides an API to manage and query any health-related measurable attributes in the system.

Monitor Service API
This API provides a REST interface to the OSGi MonitorAdmin service.
Repository Administration Service
This API is used to administer the repositories used by the system.
System Management API
A REST API that provides access to system services.
Task Service

The TaskService provides access to administration tasks that involve some form of workflow.
