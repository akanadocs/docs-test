External Key Store (com.soa.keystore.external)

Configuration of an external pki key store

The provider name of the key store.

The key store type.

The key Store Location

The Key Store Password

Encrypted Keys Returned (com.soa.keystore.external.encrypted)

Indicates whether the key store will return encrypted or unencrypted keys.

Default: false
