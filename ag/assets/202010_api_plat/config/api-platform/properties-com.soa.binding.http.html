HTTP Binding Component (com.soa.binding.http)

Configuration of the HTTP binding components

Enable Virtual Host Endpoint Selection (http.in.binding.virtualhost.endpoint.selection.enabled)

Enable/Disable Virtual Host handling when selecting endpoints in Http Binding.

Default: false

Enable Accept Header Generation (http.binding.config.generateAcceptHeader)

Enable/Disable generation of Accept headers based on the WSDL in the outgoing binding

Default: true

Operation Selection Log Message (operation.selection.error.logger.message)

The message to use when logging operation not found errors. There is a separate logger for these erors.

Default: HTTP Request Error

Operation Selection Log Category (operation.selection.error.logger.category)

The log category to use when logging operation not found errors. There is a separate logger for these errors. Errors are logged at WARN level.

Default: http.request.error
