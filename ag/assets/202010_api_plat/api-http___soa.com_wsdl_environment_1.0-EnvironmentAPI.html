Environment API

Environment API

{ http://soa.com/wsdl/environment/1.0 } EnvironmentAPI

The Environment service provides operations for finding information about the configured environments.

GET /environments getEnvironments

Returns environments within the given scope. The scopes are “global” (all environments), “local” (only those defined in the local system/tenant), and “default” (default local environment). The scope parameter is optional, the default value is “global”.

Request:

GET /environments?Scope=local HTTP/1.1
Accept: application/json

Response:

HTTP/1.1 200 OK

{
  "Environment" : [ {
    "Name" : "Development",
    "CurrentEnvironment" : true
  }, {
    "Name" : "QA",
    "CurrentEnvironment" : false
  }, {
    "Name" : "Staging",
    "CurrentEnvironment" : false
  } ]
}
