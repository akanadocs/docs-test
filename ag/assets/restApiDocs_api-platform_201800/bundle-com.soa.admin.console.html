Akana Admin Console com.soa.admin.console/2018.0.4

The Admin Console is used to manage the features and configuration of an Akana container instance. It provides a simple browser-based user interface that allows an administrator to control most aspects of the container including all configuration properties, access to remote and local repositories, addition and removal of product features, and the ability to update installed features.
