Organization Management API

{urn:com.soa.rest.services.orgmgmt}OrgMgmtREST

This API is used to manage PM organizations.

POST /organizations Add a new Organization

Add a new Organization under a given parent.
For brevity, the following Request creates an Organization with minimal parameters.
Can add Organization Contact, corresponding email, phone and address and also any
Operational/QoS/Compliance policies using this request.

Request :
POST /rest/organizations HTTP/1.1
