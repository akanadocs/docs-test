SimpleScheduler (com.soa.scheduler)

General Simple scheduler configuration

Simple Scheduler Debug (simple.scheduler.debug)

Should print debug information about executed job to System.out

Default: false

Disable scheduled Jobs (simple.scheduler.enabled)

Stops scheduled jobs from running

Default: true

Minimum number of threads (simple.scheduler.minThreads)

The Minimum number of threads

Default: 2

Minimum number of threads (simple.scheduler.maxThreads)

The maximum number of threads

Default: 10

number of seconds to keep idle threads in the pool (simple.scheduler.threadKeepAliveTime)

The number of seconds to keep idle threads in the pool

Default: 60
