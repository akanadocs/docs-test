Lifecycle Repository Configuration (com.soa.repository)

Lifecycle Repository properties

Context Root (repository.config.contextRoot)

The Context Root for the repository.

Default: /lm

Default Admin User (repository.config.defaultAdminUser)

The default Admin user for libraries created in the repository.

Default From Email (repository.config.defaultFromEmail)

The default from email for messages sent from the repository.

Super User Password (repository.config.superUserPassword)

The Super User password.

Configuration Downloads URL (repository.config.downloads)

The configuration download URL for this installation.

Default Console URL (repository.config.defaultConsoleURL)

The URL for the LM console if not set specifically for a library.

Enable Timers (repository.config.enableTimers)

Indicates that the LM timer handler should run configured timers. For example the LM message handler.
