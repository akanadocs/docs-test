Authorization Client (com.soa.auz.client)

Configuration of the authorization client

Cache Timeout (cached.auz.decision.service.cacheTimeout)

The time in seconds authorization decisions will be cached/refreshed

Default: 60

Cache Timeout (cached.auz.decision.service.expirationTimeInSeconds)

The time in seconds in which authorization decisions will be expired if not used even once

Default: 1800
