JWKS (com.akana.security.jose.profile.jwks)

JWKS Cache configuration properties

refreshTime (jwks.provider.config.cacheRefreshTime)

The refresh time in seconds the cache will be refreshed.

Default: 300

expirationPeriod (jwks.provider.config.cacheExpirationPeriod)

The expiration period in seconds the cache will be expired.

Default: 3600
