Search (com.soa.search)

Search configuration properties

Search Root Path (search.root.path)

The root context path for all search services

Default: search

Enable/Disable Search (com.soa.search.enabled)

property to enable/disable search functionality

Default: true

Max Segment Size (com.soa.search.index.merge.maxSegmentSize)

This refers to MAX_MERGE_MB property in lucene. It refers to the max size of segments selected for merging (not the max size of the resulting merged segment)

Default: 10000000

Max Indexes (com.soa.search.scheduler.process.maxIndexes)

Max number of indexes to be processed per schedule run

Default: 1000

Optimizer Index Threshold (com.soa.search.scheduler.process.optimizerIndexThreshold)

The threshold to run the index optimizer. The scheduler runs the index optimizer once it reaches processing threshold number of indexes.

Default: 1000

Sync Max Objects (com.soa.search.scheduler.process.syncMaxObjects)

Max number of objects to be loaded per object type in one schedule run to sync indexing with the system data

Default: 500

Enable Reindexing (com.soa.search.scheduler.process.reindexingEnabled)

property to enable/disable reindexing functionality

Default: true

Reindex Max Objects (com.soa.search.scheduler.process.reindexMaxObjects)

Max number of objects to be loaded per object type in one schedule run for reindexing

Default: 1000
