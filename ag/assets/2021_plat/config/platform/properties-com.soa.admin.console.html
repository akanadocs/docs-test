Admin Console Configuration (com.soa.admin.console)

Configuration of Admin Console Properties

Localhost only (admin.console.localhost.only)

Valid only when admin.console.access.restricted property is set to true. Restricts access to admin console from localhost only.

Default: false

Access Restricted (admin.console.access.restricted)

Restricts access to admin console to ports defined in system.properties.

Default: false

Admin Console Domain Enabled (admin.console.domain.enabled)

When false hides Admin Console Domain from Login screen of Admin Console.

Default: true

Basic Authentication Requirement (admin.container.basicauth.enabled)

Sets the requirement for HTTP basic auth when accessing administration URLs

Default: true

UI Theme (admin.console.theme)

Select the Admin Console UI theme

Default: legacy
