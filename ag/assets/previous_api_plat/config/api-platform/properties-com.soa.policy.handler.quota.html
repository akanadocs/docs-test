Realtime QoS Policy Handlers (com.soa.policy.handler.quota)

Configuration of the real time QoS policy handlers

Alert Checking Interval (sla.alerts.listener.trigger.repeatInterval)

The interval in milliseconds between checks for service level policy alerts

Default: 5000

Alert Checking Start Delay (sla.alerts.listener.trigger.startDelay)

The time in milliseconds before alert checking will begin when the bundle is started.

Default: 5000

Enable Refresh (sla.alerts.listener.job.enabled)

Enable or disable the alert checking job. If the alert checking job is disabled, Service Level Enforcement policies will not function properly.

Default: true
