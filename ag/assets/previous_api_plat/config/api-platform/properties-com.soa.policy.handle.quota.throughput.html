ThroughPut Policy Handler (com.soa.policy.handle.quota.throughput)

Configuration of the throughput policy handler

Counter Skip Interval (counter.skip.interval)

Number of count to skip before calling the clustered counter service

Default: 0

Cluster Cache Idle Time (cluster.cache.idle.time)

Idle time of Clustered Cache CheckIn in minutes

Default: 120

Request Thread maximum blocking time (blocking.thread.max.time)

Max amount of time in ms for which the request thread would be blocked. The thread would be blocked only when the queue size is greater than 0 in Throughput Policy.

Default: 10000

Use Clustered Cache and Clustered counter service (use.clustered.cache)

Setting to use clustered cache and clustered counter service

Default: false
