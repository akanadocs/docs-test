Spnego Policy (com.soa.policy.handler.spnego)

Configuration of the spnego policy handler

SPNEGO Consumer Subject Category (spnego.consumer.wsphandler.factory.defaultSubjectCategory)

Subject category of the identity to use for downstream SPNEGO authentication. This setting is on the policy handler factory so changes won't be reflected until new policies are constructed or reattached. It is recommended to restart the container to see consistent results.

Default: urn:org:federatedgovernance:security:subject-category:enduser

SPNEGO Provider Subject Category (spnego.provider.wsphandler.factory.defaultSubjectCategory)

Subject category of the identity to assign identities authenticated using SPNEGO. This setting is on the policy handler factory so changes won't be reflected until new policies are constructed or reattached. It is recommended to restart the container to see consistent results.

Default: urn:org:federatedgovernance:security:subject-category:enduser
