GraphQL Binding Component (com.akana.graphql.binding)

Configuration of the GraphQL binding components

Virtual Host Endpoint Selection (graphql.in.binding.component.virtualhostEndpointSelectionEnabled)

Enables/disables virtual host handling when selecting endpoints in GraphQL binding.

Default: true

GraphQL Max Message Size (bytes) (graphql.in.binding.component.graphqlPayloadMaxSizeBytes)

Sets a maximum message size, in bytes, for a GraphQL request, as a security precaution.

Default: 524288
