Paging Policy (com.soa.policy.handler.paging)

Configuration of the paging policy handler

index parameter (paging.policy.factory.indexParam)

Query parameter for specifying the index of the item to start with. This setting is on the policy handler factory so changes won't be reflected until new policies are constructed or reattached. It is recommended to restart the container to see consistent results.

Default: index

count parameter (paging.policy.factory.countParam)

Query parameter for specifying the number of the items to return. This setting is on the policy handler factory so changes won't be reflected until new policies are constructed or reattached. It is recommended to restart the container to see consistent results.

Default: count
