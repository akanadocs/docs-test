PM Facade (com.akana.facade.pm)

PM Facade Configuration

Enable/Disable Service Cache (service.lookup.cache.enable)

Enable/disable the service lookup cache

Default: false

Service Cache Size (service.lookup.cache.config.maxEntries)

Maximum number of entries in the service cache

Default: 1000

Enable/Disable Service Lookup (service.lookup.enable)

Enable/disable service lookups

Default: false

Enable/Disable Service PKI (service.lookup.pki.enable)

Enable/disable inclusion of PKI information in service documents

Default: true
