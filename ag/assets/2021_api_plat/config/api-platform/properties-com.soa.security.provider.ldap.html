LDAP Security Provider (com.soa.security.provider.ldap)

Configuration properties for the LDAP Security Provider

Cache Refresh Interval (connection.refresh.job.repeatInterval)

The interval in milliseconds to run the refresh job that evicts idle connections.

Default: 300000

Cache Refresh Delay (connection.refresh.job.startDelay)

The delay in milliseconds before the refresh job is started.

Default: 30000
