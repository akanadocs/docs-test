Container State Reporting API

{http://soa.com/uri/products/ContainerStateService/wsdl/1.0}ContainerStateREST

This API is used to report container state to Policy Manager.

PUT /containers/{key}/state Report State

Reports the state of the specified container. The following example illustrates reporting container 24141ffe-8954-4adf-9a2f-30b2054c has started.

Request:
PUT /rest/containers/24141ffe-8954-4adf-9a2f-30b2054c/state HTTP/1.1
"Started"

Response:
HTTP/1.1 200 OK
