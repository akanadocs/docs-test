SAML (com.soa.saml)

SAML Configuration

SAML Expiration (com.soa.saml.assertion.expiration)

SAML Assertion Expiration time in minutes.

Default: 240

SAML Signature Algorithm (com.soa.saml.assertion.signature.algo)

Default signature algorithm used signing SAML assertions.

Default: http://www.w3.org/2000/9/xmldsig#rsa-sha1

SAML C14N Algorithm (com.soa.saml.assertion.c14n.algo)

Default canonicalization algorithm used signing SAML assertions.

Default: http://www.w3.org/2001/10/xml-exc-c14n#WithComments
