Index Deletion Policy type (com.soa.compass.settings)

IndexDeletionPolicy which allows to keep old Lucene commit points relevant for a certain parameter (such as expiration time or number of commits), ExpirationTime is used.

Index Deletion Policy Expiration Time in seconds (compass.engine.store.indexDeletionPolicy.expirationTimeSeconds)

IndexDeletionPolicy which allows to keep old Lucene commit points relevant for a certain parameter (such as expiration time or number of commits), ExpirationTime is used.

Default: 300

Optimizer Schedule (compass.engine.optimizer.schedule)

Optimizer that will be used to optimizer the search engine index true if the optimizer is going to be scheduled, false otherwise.

Default: true

Optimizer schedule period (compass.engine.optimizer.schedule.period)

Optimizer schedule period Determines the how often the optimizer will kick in (in seconds).

Default: 600

deleteMarkDeletedDelta (compass.engine.store.jdbc.deleteMarkDeletedDelta)

Setting that deletes the entries that are marked as deleted delta (the number of milliseconds that is set here) time ago

Default: 600000

Merge factor (compass.engine.mergeFactor)

Merge factor defines how often the segments will be merged. Default is 10

Default: 10

Jdbc Lock Type (compass.engine.store.jdbc.lockType)

jdbc lockType determines how index is locked to avoid multiple index writers trying to update index at the same time.

Default: com.soa.search.compass.SOAPhantomReadLock
