HTTP Binding Component (com.soa.binding.http)

Configuration of the HTTP binding components

Enable Virtual Host Endpoint Selection (http.in.binding.virtualhost.endpoint.selection.enabled)

Enable/Disable Virtual Host handling when selecting endpoints in Http Binding.

Default: false

Enable Accept Header Generation (http.binding.config.generateAcceptHeader)

Enable/Disable generation of Accept headers based on the WSDL in the outgoing binding

Default: true
