WSDL Management (com.soa.wsdl)

WSDL Management configuration

Maximum WSDL Documents (wsdl.storage.manager.maxEntries)

Defines how many WSDL documents can be held in memory at any given time. A value of -1 indicates an unlimited number will be held in memory.

Default: -1

Treat Restore as Error (wsdl.storage.manager.errorOnRestore)

Indicates whether to report restoring WSDL documents from disk as an error or simply trace it.

Default: false

Enable WSDL Cache (wsdl.storage.manager.enabled)

Globally enable/disable the WSDL caching facility.

Default: true
