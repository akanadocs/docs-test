Contract Management API

{http://soa.com/uri/products/ContractsManager/wsdl/1.0}ContractAdminService

The API used to manage PM contracts

GET /contracts Returns contracts for the given criteria.

Returns contracts based on given provider service or provider organization or consumer organization. A usage example is as follows:

Request:

GET /rest/contracts?providerOrg=uddi:f005b0b2-1079-11e5-a330-a83ecc4e96f2  HTTP/1.1

Response:

		{"ContractInfo": [{
		   "Description": "Created Contract5",

		      "StartDateTime":       {
		         "DateTime": "2015-04-23T22:36:00-07:00",
		         "TimezoneName": "America/Los_Angeles"
		      },

POST /contracts Creates a contract

Creates a contract in the given organization with given parameters. A usage example is as follows:

Request :

POST /rest/contracts HTTP/1.1
 {
    "ContractInfo": {
      "Name": "CreatedContract1",
      "Anonymous": "true",
      "State": "Active",
      "Description": "Created Contract1",
      "ProviderOrganizationKey": "uddi:f005b0b2-1079-11e5-a330-a83ecc4e96f2",
      "EffectiveDateTimes": {
        "StartDateTime": {
          "DateTime": "2015-04-23T22:36:00-07:00",
          "TimezoneName": "America/Los_Angeles"
        },
        "EndDateTime": {
          "DateTime": "2016-04-22T22:36:00-07:00",
          "TimezoneName": "America/Los_Angeles"
        },
        "Expire": "false"
      }
    },
    "AddContractContext": {
      "Action": "@CreateOffer",
      "Comments": "My Comments"
    }
 }

Response:

HTTP/1.1 201 Created

Request:

PUT /contracts/{key}/workflow Perform Workflow Action

Performs a workflow action on the contract with the given key. The available workflow actions are based on the current workflow process in use for contracts in the system. A usage example is as follows:

Request:
PUT /rest/contracts/d92830eb-1125-11e5-93d5-eb4e298f7990:1043/workflow HTTP/1.1
{
"Action" : "Activate Contract",
"Comment" : "Putting contract into production.",
"UserName" : "jdoe"
}

Response:
HTTP/1.1 200 OK
