OAuth (com.soa.auth.pingfederate)

OAuth resource server configuration properties

Use Bearer Token Long Form ()

Property to disable/enable long tokens for Bearer token type. Short tokens will be used if this property is set to false which is default.

Refresh Time (com.soa.auth.pingfederate.grant.refreshTime)

The refresh time in seconds the cache will be refreshed.

Default: 300

Expiration Period (com.soa.auth.pingfederate.grant.expirationPeriod)

The expiration period in seconds the cache will be expired.

Default: 3600

Cache Max Capacity (com.soa.auth.pingfederate.grant.cacheMaxCapacity)

Property indicating the number of maximum entries that can be cached at any time.

Default: 100000
