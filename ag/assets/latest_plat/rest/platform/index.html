Public APIs for Release 2020.2.3

Endpoints

Bundle Administration Service { urn:soa.com:/uri/products/admin/bundleservice/wsdl/1.0 } BundlesService

API used to manage system bundles.

Configuration Admin API { urn:soa.com:/uri/products/configservice/wsdl/1.0 } ConfigurationService

A REST API that provides access to the Configuration Admin service.

Feature Administration Service { urn:com.soa.admin.service.features.jaxrs } FeatureService

This API is used to administer the installation,removal, and resolution of features.

Health Service { urn:com.soa.admin.health.jaxrs } JAXRSHealthService

The Health Service provides an API to manage and query any health-related measurable attributes in the system.

Monitor Service API { urn:com.soa.admin.service.monitoring.jaxrs } JAXRSMonitorService

This API provides a REST interface to the OSGi MonitorAdmin service.

Repository Administration Service { urn:com.soa.admin.service.obr.jaxrs } RepositoryService

This API is used to administer the repositories used by the system.

System Management API { urn:com.soa.admin.service.system.jaxrs } SystemService

A REST API that provides access to system services.

Task Service { urn:com.soa.admin.service.tasks.jaxrs } TaskService

The TaskService provides access to administration tasks that involve some form of workflow.
