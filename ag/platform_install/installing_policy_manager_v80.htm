Introduction

The Policy Manager solution is composed of a base set of services that perform key service management processes, and one or more Containers that perform the management and monitoring of web services. It includes a Management Console for configuring and maintaining your Governance solution, and provides a variety of wizard utilities for performing configuration and maintenance activities.

Step 1: Install Akana Platform

The Policy Manager zip file must be installed to the Akana Platform. See Installing the Akana Platform for instructions.

If you have previously installed the Akana Platform, skip to the next step.

Step 2: Download Policy Manager Zip File

Download an Policy Manager self-extracting zip file from the Akana Customer Support Site:

  1. Launch www.support.akana.com
  2. Go to Downloads > PolicyManager > 8.0.
  3. Download a platform zip file:

    akana-pm_8.0.xxxx.zip
