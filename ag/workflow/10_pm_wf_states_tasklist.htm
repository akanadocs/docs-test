Special Workflow States and Workflow Tasks List

Description of special workflow state and workflow task list.

Introduction

The Workflow states can be classified into two categories:

  • A state that requires immediate attention
  • A state that does not

To mark a workflow state as requiring immediate attention, you can prepend or append one or more exclamation point symbols (!) to the name of the Workflow state. For example, Pending Approve!!! or !Pending Approve.

When a workflow enters a state with this special markup, it is added into the Workflow task list so user can be immediately notified via Workbench's Workflow task portlet.
