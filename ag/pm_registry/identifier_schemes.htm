Identifier Schemes

Define your own custom identifier schemes that align with your specific business requirements.

To view the Identifier Schemes Summary page, go to Configure > Registry and click the Identifier Schemes tab.

Table of Contents

  1. Add Identifier Scheme
  2. Modify Identifier Scheme
  3. Delete Identifier Scheme

Add Identifier Scheme

Add an identifier scheme by defining its Name and Description. The naming convention must conform to one of the Uniform Resource Identifier (URI) schemes.

  1. Go to Configure > Registry.
  2. Click the Identifier Schemes. The Identifier Schemes Summary page is displayed.
  3. Click the Add Identifier Scheme tab. The Add Identifier Scheme page is displayed.
  4. (Optional) Enter a Key (UDDI key name). If you don't specify the key, it is generated by the Registry Manager Subsystem.
  5. Enter the Name of the new identifier scheme. Must conform to any Uniform Resource Identifier Scheme.
  6. Enter a Description for the new identifier scheme. The description is displayed on the Identifier Summary.
  7. Click Apply. The system saves the new definition and returns to the Identifier Schemes Summary.

Modify Identifier Scheme

Note: Changing the underlying properties of an identifier scheme after it has been used to associate an identifier with an entity will automatically update the associations.

  1. In Policy Manager, go to Configure > Registry.
  2. Click the Identifier Schemes tab. The Identifier Schemes Summary page is displayed.
  3. Click Modify Identifier Scheme. The Modify Identifier Scheme page is displayed.
  4. Update one or more of the following:
    • Name—The name of the identifier scheme. The naming convention used must conform to any Uniform Resource Identifier Scheme.
    • Description—Description of the identifier scheme. The description is displayed on the Identifier Summary.
  5. Click Apply. The system saves the new definition and returns to the Identifier Schemes Summary.

Back to top

Delete Identifier Scheme

  1. Go to Configure > Registry.
  2. Click the Identifier Schemes tab. The Identifier Schemes Summary page is displayed.
  3. Click Delete Identifier Scheme.
  4. At the confirmation message, click OK. The Identifier Scheme is permanently removed from the Category Hierarchy.
