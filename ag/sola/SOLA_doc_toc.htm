SOLA Documentation Summary

A summary of documentation PDF files available for the SOLA product.

SOLA 6.4.2 Developer User's Guide

Learn about the SOLA Developer, a browser-based IDE that you can use to create, manage, secure, and test services.

SOLA REST User Guide

Learn about the SOLA REST API Designer, an easy way to prepare your SOLA SOAP-based services so that they can be accessed via REST.

SOLA Resource Manager User's Guide

Learn about Resource Manager, a runtime governance tool for managing SOLA and controlling SOLA's runtime authentication and authorization to resources running in specific SOLA containers.

SOLA DOM API Overview

Learn about how to use the SOLA DOM API, which implements the functionality of DOM in a format that is easily used by procedural languages like COBOL, to inquire on, modify, and create XML documents.

SOLA DOM API Reference Guide

A reference guide for users of the SOLA DOM API, which implements the functionality of DOM in a format that is easily used by procedural languages like COBOL, to inquire on, modify, and create XML documents.

SOLA Administration Guide

A guide to SOLA 6.4.2 for the system administrator.

SOLA/LM Integration User's Guide

Learn about SOLA Development Studio integration with Lifecycle Manager.

SOLA 6.4 Migration Guide

Learn about conversion utilities to migrate from the SOLA 5.1 DB2 database to the SOLA 6.1 XML database.

SOLA Installation Guide

Learn how to install SOLA.

SOLA Prerequisites Guide

Learn about system requirements and prerequisites for installing and running SOLA 6.4.2.

SOLA Error Messages Guide

Learn about the error messages returned by SOLA, including explanations and recommended actions.
