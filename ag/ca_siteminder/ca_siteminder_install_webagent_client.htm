Step 3: Install CA SiteMinder Web Agent on Client Machines

Learn how to install CA SiteMinder Web Agent on Client Machines.

Using Admin Console Integrate CA SiteMinder with Policy Manager (Main Topic)

Install CA SiteMinder WebAgent on Client Machines

To install the web agent on client machines, launch the CA SiteMinder Web Agent v6QMRS Wizard and perform the following steps on all Policy Manager and Network Director SOA Container instances.

  1. Launch the CA SiteMinder Web Agent installer. For example, on Windows systems, run the installer by launching the nete-wa-6qmr5-cr035-win32.exe program. (The installer for each release of the Web Agents will have a different but similar name.) The Introduction screen displays.

  2. Review the introduction and click Next to continue.


    CA SiteMinder Web Agent v6QMR5: Introduction

  3. The License Agreement screen displays. Review the license agreement. If you agree to the terms, select the "I accept the terms of the License Agreement" radio button and click Next to continue.


    CA SiteMinder Web Agent v6QMR5: License Agreement

  4. The Important Information screen displays. Review the notes and click Next to continue.


    CA SiteMinder Web Agent v6QMR5: Important Information

  5. The Choose Install Path screen displays. Enter the install path and click Next to continue.


    CA SiteMinder Web Agent v6QMR5: Choose Install Location

  6. The Choose Shortcut Folder screen displays. Enter your shortcut preference and click Next to continue.


    CA SiteMinder Web Agent v6QMR5: Choose Shortcut Folder

  7. The Pre-Installation Summary screen displays. Review the installation summary and click Next to continue.


    CA SiteMinder Web Agent v6QMR5: Pre-Installation Summary

    Note: If the following pop-up message (or similar) displays, SOA Software recommends selecting the "No to ALL" button.


    CA SiteMinder Web Agent v6QMR5─Pre-Installation Summary (Overwrite Existing File Message)

  8. The Web Agent Configuration screen displays. Configuring a Web Agent is not required. Select the "No, I would like to configure the Agent later" radio button and click Next to continue.


    CA SiteMinder Web Agent v6QMR5─Web Agent Configuration

  9. The Install Complete screen displays. Restarting the system is not required. Click the "No, I will restart my system myself" radio button. Click Done to complete the configuration and exit the wizard.


    CA SiteMinder Web Agent v6QMR5: Install Complete
