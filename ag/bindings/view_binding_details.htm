View Binding Details

Learn how to view elements of a Binding definition including Key, QName, Type, Description, and Number of References.

To view binding details

  1. Log in to the Policy Manager Management Console.
  2. Go to Configure > Registry > Bindings.
  3. On the Bindings Summary screen, select a binding line item and select View Binding Details from the Actions drop-down list. The Binding Details screen is displayed.
