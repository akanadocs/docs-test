This guide provides instructions for installing the SOA Software Platform 7.1 GA, SOA Software Platform Updates, configuring a Container Instance, installing Policy Manager Features using the SOA Software Administration Console, and performing administration tasks.

This technical note provides instructions for upgrading Policy Manager 7.0 (PM70) to Policy Manager 7.1 (PM71), and upgrading Policy Manager 6.1 (PM61) to Policy Manger 7.1 (PM71).
