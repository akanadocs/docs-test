Akana Intermediary for Microsoft with Akana Platform 2020.2.4

The Intermediary for Microsoft product relies on a compatible version of the Akana Platform, including compatible versions of all applicable resources such as database types and versions, hardware, and operating system software.

For details of the system requirements for Akana Platform 2020.2.4, see System Requirements for Akana Platform 2020.2.x.

This document provides supplementary information on recommended versions that Intermediary for Microsoft was tested with for Akana Platform 2020.2.4.

Prerequisites

Before you install the Intermediary for Microsoft product, you must have the following:

  • Akana Policy Manager installed
  • A relational database
  • A Windows Server machine
