Akana Intermediary for Microsoft with Akana Platform 2020.2.x

Valid in Version: 2020.2.x

Intermediary for Microsoft version 2020.2.x is valid for use with Microsoft Internet Information Server (MSIIS) 8.5, with .NET framework 4.8.

For more information on system requirements, see System Requirements for Intermediary for Microsoft 2020.2.x.

Known limitations

Most aspects of Intermediary for Microsoft work exactly the same in version 2020.2.x as in earlier published and documented versions. However, the items below are limitations in this version:

  • The Akana Intermediary Support Service OAuth feature does not work correctly in version 2020.2.x. If you need this feature, use an earlier version of Intermediary for Microsoft. {Query_01: do we need to say this or no?}
  • Intermediary for Microsoft does not currently support writing directly to a MongoDB database. {Query_02: do we need to say this or no?}
  • If you are using an audit policy, such as the WS-Auditing Message Policy policy, do not include the Audit Transport Headers setting. This setting currently results in a Windows error. {Query_03: do we need to say this or no?}
