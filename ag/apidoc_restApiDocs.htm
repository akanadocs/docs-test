REST API Documentation

Akana API Platform API Documentation

The following generated REST API documentation sets are available for the Akana API Platform and Policy Manager:

  • Akana API Platform REST APIs version 2018.0.5 (includes Policy Manager and Community Manager generated API doc)
  • Akana API Platform REST APIs version 8.4.34 (includes Policy Manager and Community Manager generated API doc)
  • Policy Manager REST APIs, version 8.2.35

The generated documentation for your version is also available in the /docs folder of your installation.

You can also view the authored API doc for developer portal APIs: Akana API Platform API: Overview.

Akana Platform API Documentation

The following generated REST API documentation is available for the Akana Platform:

  • Akana Platform REST APIs version 2018.0.4

The generated documentation for your version is also available in the /docs folder of your installation.

The Akana Platform API documentation includes the following services:

  • Bundle management

    Install, uninstall, update, start, and stop bundles

  • Feature management

    Find, resolve, install, uninstall, update, and roll back features

  • Configuration service

    Config Admin CRUD, property metadata

  • Repository management

    OBR Admin, discover and resolve resources

  • Task service

    Query and execute configuration tasks

  • System admin

    System information, restart, and shutdown
