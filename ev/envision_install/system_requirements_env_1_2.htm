System Requirements for Akana Envision 1.2

The following table lists the minimum system requirements for running Akana Envision 1.2 on Windows and UNIX platforms.

Note: Envision 1.2 runs on Akana Platform 8.2.x. If you're using Akana Platform 8.4.x, use Envision 1.2.1. see System Requirements for Akana Envision 1.2.1.

Updated: January, 2018

MongoDB
