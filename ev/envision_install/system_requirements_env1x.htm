System Requirements for Akana Envision

The following table lists the minimum system requirements for running Akana Envision on Windows and UNIX platforms.

Updated: September 30, 2015

Supported Platforms: 1.0, 1.1

 

Component Name Requirement
MongoDB 3.0.0

Version 3.0.0. Windows 64-bit (MSI), Linux 64-bit (TGZ), and Solaris 64-bit (TGZ) are supported.

See MongoDB Downloads website to download a MongoDB Version 3.0.0 file for your platform.

Akana Platform

Version 8.0 (for Envision 1.0), Version 8.1 (for Envision 1.1).

See Installing Akana Platform for download and installation instructions

Policy Manager
(for Envision/Policy Manager Integration only)

Version 8.1.

See Performing Policy Manager / Envision Integration Installation for download and installation instructions.
