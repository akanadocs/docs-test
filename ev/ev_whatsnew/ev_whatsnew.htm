Changes in the Envision 2020.1.1 Release

The Envision 2020.1.1 release is a major update. Envision is now compatible with the latest version of the Akana API Platform (2020.1.1).

For details about the key features and enhancements in 2020.1.x, see Akana Release Notes.

In version 2020.1.1, the Operational Metrics Policy, with its default settings that populated the API Operational Metric Dataset, is deprecated. The policy and the dataset both currently appear in the user interface, but will be removed in a later version. We recommend that you do not use this dataset and its associated policy.

Instead, use the Business Metrics Policy, which has much more flexibility for configuring the parameters of your dataset.

For guidelines and recommendations regarding how to optimize your data parameters and dataset so that you get the information you need while maintaining optimal performance, refer to these two new documents:

  • Dataset Building Guide for Envision
  • Chart Building Guide for Envision

For system requirements, see System Requirements for Akana Platform 2020.1.x. For Envision, MongoDB is required.
