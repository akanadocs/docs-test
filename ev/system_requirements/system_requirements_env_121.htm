System Requirements for Akana Envision 1.2.1

The following table lists the minimum system requirements for running Akana Envision 1.2.1 on Windows and UNIX platforms.

Note: Envision 1.2.1 runs on Akana Platform 8.4.x. To use the latest version of Envision, 2018.0.0, see System Requirements for Akana Envision 2018.0.0.

Updated: November, 2018

Component Name Requirement
MongoDB

Supported versions: 3.2.12, 3.2.13, and later versions up to 3.2.18, Windows 64-bit (MSI) and Linux 64-bit (TGZ).

To download a MongoDB Version 3.2.18 file for your platform, refer to the MongoDB Downloads website.
Akana Platform

Version 8.4

For download and installation instructions, see:

  • Installing the Akana API Platform 8.4x
Policy Manager
(for Envision/Policy Manager Integration only)		 Version 8.4
