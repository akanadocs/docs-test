Deprecation Notices: 2020.1.5

Some model object properties are deprecated and will be removed in 2020.2.0

In version 2020.2.0, some policies will be removed from several of the model objects used by existing developer portal API operations. Model objects, and the operations affected, are listed below. Please adjust your automation in advance of upgrading to 2020.2.0.

ApiVersion:

  • GET /versions/{ApiVersionID} (http://docs.akana.com/cm/api/apis/m_apis_getAPIVersion.htm)
  • POST /{ApiID}/versions (http://docs.akana.com/cm/api/apis/m_apis_createAPIVersion.htm)

TargetAPI:

  • TargetAPI (field TargetAPI inside ApiVersion; see above)

APIImplementation:

  • GET /versions/{ApiVersionID}/implementations/{ImplCode} (http://docs.akana.com/cm/api/apis/m_apis_getAPIImplementation.htm)
  • PUT /versions/{ApiVersionID}/implementations/{ImplCode}/deployments (http://docs.akana.com/cm/api/apis/m_apis_modifyAPIImplementationDeployments.htm)
  • DELETE /versions/{ApiVersionID}/implementations/{ImplCode} (http://docs.akana.com/cm/api/apis/m_apis_deleteAPIImplementation.htm)
