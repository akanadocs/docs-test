Key Features: 2020.2.0

Note: The key features here are specific to 2020.2.0 and are not available in earlier 2020.1.x update releases. For features and enhancements also available in 2020.2.0 but delivered in previous 2020.1.x update releases, see each update version below.

Technology upgrades

This release incorporates several upgrades to technologies and tools:

  • Jetty updated to version 9.4.31
  • Mongo updated to version 3.6.16
  • Elasticsearch updated to version 7.8.x
  • Adobe Flash replaced in Policy Manager's Real Time and Historical Charts
AWS CloudHSM support

Akana adds support for the AWS CloudHSM cloud-based hardware security module.

Docker images support

Akana can now be installed via Docker images which will enable better environment standardization, portability, and improved compatibility and maintenance.

NTLM V2 support

The authentication protocol NT LAN Manager version 2Â (NTVMv2) is now supported.

New JOSE Profile-Driven Security Policy

This release adds a new policy, JOSE Profile-Driven Security, which can use existing profiles as a template. The new policy supports the RSA Adaptive Authentication for eCommerce, the Visa Token Service, and the UK Open Banking Event Notification API, in addition to other features. See [link to docs coming] on the Akana documentation website for details.

Developer portal enhancements

Multiple enhancements have been made to the developer portal. Among them are:

  • Bonita theme Improvements: Bonita theme now includes an API Access Wizard, supports authored documentation, and has more intuitive navigation functionality.
  • Operation-specific policy support. Policies can now be assigned to a specific operation in an API implementation, rather than the entire implementation.
  • Customization documentation improvements [Ranga, what are these?]
Envision enhancements

Envision has been enhanced with several usability improvements and security fixes, including the addition of chart creation guidance when filtering, the display of chart loading information, and the ability to edit a chart without first previewing it, improving performance.

Lifecycle Manager Repository Client

The Lifecycle Manager Repository Client is now available on macOS.
