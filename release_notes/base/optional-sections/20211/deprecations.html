Deprecations and Removals

Existing Akana Administration Console to be deprecated in a future release

The new Akana Administration Console is automatically installed with this release, although it is not yet the default. The new Akana Administration Console will become the default in a future release, and the old console UI will be deprecated and renamed "Akana Administration Console Legacy," then it will be removed from the product.

Support ticket: No support ticket

Apache Axis1.4 deprecated and replaced by Axis2 1.7.9

Apache Axis1.4 is deprecated with this release, replaced by Apache Axis2 1.7.9. Axis is a call component used in the Management Point pipeline.

Support ticket: No support ticket
