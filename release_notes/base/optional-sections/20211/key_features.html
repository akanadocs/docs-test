GraphQL query language now supported

This release adds support for GraphQL, a query language for APIs. Using GraphQL, the client can request information in a single GraphQL request that might previously have required multiple traditional REST API requests. For detail, see "Installing and Configuring GraphQL for the Akana API Platform" on the Akana docs site.
