System Requirements for Akana Platform 8.4x

The following table lists the minimum system requirements for running Akana Platform 8.4x.

For instructions for installing or upgrading, see Installing the Akana API Platform 8.4x or Upgrading Akana API Platform from Version 8.2 to Version 8.4x.

Supported Platforms: 8.4x

Name Requirement
Akana Platform Host

Hardware:

Single CPU, 2Ghz, 2GB RAM

Operating Systems:

  • Windows - Windows Server 2008, Windows Server 2012, Windows 10
  • Linux - Red Hat Enterprise Linux 5.8, 6.2, 7
Client browser for accessing developer portal

Default Theme and Hermosa Theme:

  • Firefox version 25 and later
  • Chrome version 31 and later
  • Internet Explorer versions 9, 10, and 11

Simple Developer Theme:

  • Firefox version 32 and later
  • Chrome version 38 and later
  • Internet Explorer versions 9, 10, and 11
Database versions

Akana API Platform 8.4 supports the following database versions:

  • MySQL 5.6

    Note: If you're using MySQL 5.6 with version 8.0.x or later of Policy Manager, the Akana Platform, or Community Manager, you must configure some specific MySQL property settings. See MySQL Database Property Settings.

  • Oracle 11g, 12c, and 12c Release 2 (supported in version 8.4.16 and later)

    Note: The Oracle database password must not start with a number or special character.

  • Microsoft SQL Server 2012, 2014 (supported in version 8.4.2 and later), and 2016 (supported in version 8.4.15 and later).

    Note: If you're using Microsoft SQL Server, you must set the quartz trigger property in the Akana Administration Console to True. See Database notes: Microsoft SQL Server.

The following MongoDB versions are supported:

  • 3.2.18

Note: Depending on the database type and version that you're using, there might be specific requirements, constraints, or additional steps needed. For details, refer to the Database Notes section of the installation documentation.
Memory configuration

The default maximum heap size for all Akana Containers (i.e., Java processes) is 2048 MB.

Recommended: 4096 MB.

MongoDB

Supported versions (Windows 64-bit MSI and Linux 64-bit TGZ):

  • Platform versions prior to 8.4.13: MongoDB versions 3.2.12 and 3.2.13
  • Platform versions 8.4.13 and later: MongoDB versions 3.2.12, 3.2.13, and later versions up to 3.2.18

To download a MongoDB Version 3.2.18 file for your platform, refer to the MongoDB Downloads website.
