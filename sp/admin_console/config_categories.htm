Using Configuration Categories

Learn about properties you can configure in the Administration Console.

Administration Console Configuration Categories and Properties

The docs\configAdmin folder of the Policy Manager release directory includes documentation for Configuration Categories that are available via the Configuration tab of the Akana Administration Console. The index.htm file presents a list of configuration categories and list of supported properties with a brief description of the property and default state (if applicable).

You can also view the documents for each platform version: Configuration Categories and Properties.
