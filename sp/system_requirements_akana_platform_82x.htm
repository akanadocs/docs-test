System Requirements for Akana Platform 8.2x

The following table lists the minimum system requirements for running Akana Platform 8.2x.

For instructions for installing or upgrading, see Installing the Akana API Platform 8.2x or Akana API Platform: Upgrade Guide VVersion 8.0 to 8.2 (PDF).

Supported Platforms: 8.2x

Component Name Requirement
Akana Platform Host

Hardware:

Single CPU, 2Ghz, 2GB RAM

Operating Systems:

  • Windows - Windows Server 2008, Windows Server 2012, Windows 10
  • Linux - Red Hat Enterprise Linux 5.8, 6.2, 7
Client browser for accessing developer portal

Default Theme and Hermosa Theme:

  • Firefox version 25 and later
  • Chrome version 31 and later
  • Internet Explorer version 9 and later

Simple Developer Theme:

  • Firefox version 32 and later
  • Chrome version 38 and later
  • Internet Explorer version 9 and later
Database versions

Akana API Platform 8.2 supports the following database versions:

  • MySQL 5.6
  • Oracle 11g, 12c
  • Microsoft SQL Server 2012 and 2014 (supported in version 8.4.2 and later)

The following database versions will be certified at a later date:

  • DB2 9.7 and 10.5
