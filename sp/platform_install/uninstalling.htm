Uninstalling a specific product feature in the Akana Administration Console

If you've installed a specific product feature in the Akana Administration Console, and later find that you don't want that feature, you can uninstall that specific feature in the Console.

To uninstall a feature in the Akana Administration Console
  1. Log in to the Akana Administration Console.
  2. Click the Installed Features tab.
  3. On the list of installed features, find the feature you want to uninstall.
  4. To the right of the item, select the Uninstall icon. The system uninstalls the feature, removes it from the Installed Features list, and moves it back to the Available Features list.
  5. Change the values as needed, and then click Apply Changes.

For more information on using the Akana Administration Console, see Installed Features.

Uninstalling on a Windows machine

If your installation is on Windows, you can uninstall via the Control Panel, Programs and Features.

When you uninstall, all program files are removed. However, certain configuration files, containing information you entered as part of setup, are left behind. This is intentional, since you might be uninstalling an earlier version as part of upgrading to a later version.

This section includes two procedures:

  • To uninstall the Akana API Platform
  • To remove residual installation files
To uninstall the Akana API Platform (Windows)
  1. Go to the Control Panel on your machine and choose the Programs and Features list.
  2. For example, choose Start | Control Panel | Programs and Features. The exact naming and navigation might vary on your machine.
  3. On the Programs List, find SOA Software Intermediary.
  4. Select it, and then click Uninstall.
  5. You can click the Uninstall button at the top of the list, or choose from the right-click menu.
  6. At the “Are you sure…?” prompt, click Yes.
  7. When the process is complete, close the Programs list.

Note: The uninstall process leaves behind some residual data files. This is intentional, since in some scenarios uninstallation is sometimes the first step in upgrading. If you no longer want the data files on your machine, follow the instructions below to remove them.

To remove residual installation files

Before following the steps below, uninstall SOA Software Intermediary from the Programs and Features list as explained above.

  1. Navigate to the Intermediary for Microsoft installation location on your machine. For example: 
    C:\Program Files (x86)\Akana\AAP84
  2. Delete the Akana API Platform folder. You’ll need to provide Administrator permission for the deletion.

    The files are removed. Your system is now completely clear of all residual Akana API Platform files.
