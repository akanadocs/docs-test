Introduction

This topic provides instructions for installing Policy Manager 8.0.

Step 1: Install Akana Platform

The Policy Manager zip file must be installed to the Akana Platform.

  1. See Installing the Akana Platform for instructions.
  2. If you have previously installed the Akana Platform, skip to the next step.

Step 2: Download and Install Policy Manager Zip File

Download an Policy Manager self-extracting zip file from the Akana Customer Support Site:

  1. Launch www.support.akana.com
  2. Go to Downloads > PolicyManager > 8.0.
  3. Download a platform zip file:

    akana-pm_8.0.xxxx.zip

  4. Extract the Policy Manager zip file to an existing Akana Platform folder. The directory structure will look similar to the following after the file is extracted:

Step 3: Download and Install Policy Manager Updates

Akana periodically issues updates for the Akana Platform. Update files can be downloaded from the Akana Customer Support Site. Update files must be copied to the Akana Platform Release Directory. Each update file must be extracted in version number order (earliest version first) prior to confugiring a container instance.

  1. Launch www.support.akana.com
  2. Go to Downloads > PolicyManager > 8.0 > Updates > <Update Version>.
  3. Download the update zip file(s).
  4. Extract the platform zip file(s) to the Akana Platform Release Directory in version number order.
  5. When the "Confirm file replace" dialog displays, click Yes to All. The automated zip file then updates a series of files in the release directory folders.
  6. After applying the updates, delete the /instances/configurator/cache directory if it exists.
  7. After the automated zip file completes its processing, continue to the next step to configure a container instance.
