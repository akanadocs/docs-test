Introduction

This topic provides instructions for installing Policy Manager 8.0.

Step 1: Install Akana Platform

The Policy Manager ZIP file must be installed to the Akana Platform 8.1.

  1. For instructions, see Installing the Akana Platform (Version 8.1).
  2. If you have previously installed the Akana Platform 8.1, skip to the next step.

Step 2: Download and Install Policy Manager ZIP File

Download the Policy Manager self-extracting ZIP file from the Akana Customer Support Site.

To download and install the Policy Manager ZIP file
  1. Go to the Akana Library site at https://library.akana.com/display/MAIN/Akana+Library.
  2. In the Downloads section, click API Gateway > PM 8.0.
  3. Download the Policy Manager ZIP file:

    akana-pm_8.0.xxx.zip

Step 3: Download and Install Policy Manager Updates

Akana periodically issues updates for the Akana Platform. You can download update files from the Akana Customer Support Site, and copy them to the Akana Platform Release Directory.

Note: It's important that you extract each update file in turn, following the version number sequence with the earliest version first, before configuring a container instance.

To download and install Policy Manager updates
  1. Go to the Akana Library site at https://library.akana.com/display/MAIN/Akana+Library.
  2. In the Downloads section, click API Gateway > PM 8.0.
  3. In the Updates section, download all available updates, one or more ZIP files, to the Akana Platform Release Directory.
  4. Extract the ZIP files to the Akana Platform Release Directory in version number order (earliest version first).
  5. At the "Confirm file replace" dialog, click Yes to All. This updates applicable files in the release directory folders.
  6. After applying the updates, delete the /instances/configurator/cache directory if it exists.
  7. After the automated ZIP file completes its processing, continue to the next step to configure a container instance.
