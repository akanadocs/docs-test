How do I create a discussion with my development team about my app project?

You can use the App Board to create a discussion about your app project with your team members. Only your team members have access to the Board for your specific app.

To create a discussion about an app:
  1. Navigate to My Apps > App Name.
  2. In the left menu, click Board. The App Board displays.
  3. Choose the Discussion topic type.
  4. Give the discussion a short descriptive title, and add information about what you want to discuss.
  5. Enter one or more tags to identify the discussion. Tags are used by the search feature to help users find information.
  6. Click Submit.

How do I report a problem with an API using the App Board?

You can use your App Board or API Board to submit trouble tickets directly to the Admin Team.

To report a trouble ticket through the App Board:
  1. Navigate to My Apps > App Name.
  2. In the left menu, click Board. The App Board displays.
  3. At the top, click the Ticket topic type.
  4. Click the What issues would you like to report? text box and then add the following information:
    • A short descriptive title.
    • Tags to identify the ticket. Tags are used by the search feature to help users find information.
    • Information about the issue.
    • Priority (Minor, Major, or Critical)
    • Implementation (Sandbox or Live)
    • The API the issue relates to
  5. Click Submit.

For information about managing and viewing existing tickets, see How do I filter information in the App Board?

Note: Tickets are displayed in the applicable API Board as well as in your App Board.

How do I change the priority of a problem I reported?

You can change the priority of a ticket if you created it yourself or are an administrator for the resource.

To change the priority of an API ticket through the App Board:
  1. Navigate to My Apps > App Name.
  2. In the left menu, click Board. The App Board displays.
  3. In the API drop-down, choose the API name.
  4. Find the ticket on the list, and click Edit Priority.
  5. Write a comment, choose a new priority, and click Confirm.

How do I delete an App Board item?

The App Board allows you to delete discussion or ticket post types. When you delete a post, all comments entered by site users on the post are also deleted. You can only delete posts you have personally added.

To delete an App Board item (discussion or ticket):
  1. Navigate to My Apps > App Name.
  2. In the left menu, click Board. The App Board displays.
  3. Locate the Board item you want to delete, using the filters or search feature if needed.
  4. In the upper right-hand corner of the post, click x to the right of the post icon.
  5. At the confirmation message, click OK to confirm deletion.
