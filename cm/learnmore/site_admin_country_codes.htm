Country Codes

Valid in Version: 2020.1.0 and later

Configure country codes for use in telephone number fields in the developer portal.

Table of Contents

  1. How do I configure country codes for telephone numbers?

How do I configure country codes for telephone numbers?

If you want to collect telephone numbers in user information pages in the developer portal, you'll need to add one or more country codes that you will support.

Country codes are used in certain developer portal pages relating to users, such as the Sign Up page and the Profile > Details page. They are also needed if two-factor authentication (2FA) is in use.

Country codes that you've already added are displayed on the page.

The platform only allows one entry per country code. So, for example, you cannot include United States, with a value of +1, and then Canada, with a value of +1. When you try to save the main page you'll see an error. Instead, choose the official designation from the list; in this example, United States and Canada is one entry.

Note: When you add one or more country codes, the telephone number field becomes a required field for users signing up or modifying the user profile.

To add a country code

  1. Log in as a Site Admin and go to the Admin section.
  2. Go to More > Admin > Country Codes.
  3. Click Add Country Code.
  4. In the Add Country Code overlay, add:
    • Country Name: for example, United Kingdom. Start to type, and your entry is matched against the list of valid country names/codes.
    • Country Code, starting with a + sign: for example, +44. When you get a match in the Country Name field, the Country Code is added automatically.
  5. Click Finish. The new entry is added to the list.
  6. Repeat steps 3 through 5 as needed to add more country codes.
  7. When done, click Save.

    Note: If you close the page without clicking Save, the country codes you've added are not saved.

    If needed, you can use the Reset button at any point before saving, to remove changes you just made.

To delete a country code

In the Add Country Code page, just click the X to the right of the entry on the list. Click Save in the page to make the change permanent.

Note: You cannot update a country code. If you make a mistake, just delete and re-add.
