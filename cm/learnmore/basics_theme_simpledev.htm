Simple Developer Theme

Simple Developer theme was introduced in version 7.2 of the developer platform.

This theme is a simplified user interface supporting all functions needed by App developers and most functions needed by API Admins (API documentation must be uploaded by the Site Admin in Hermosa Theme (up to 2019.0.x).
