Default Theme

Note: Default Theme is deprecated in Version 2020.1.0.

Default theme was the version of the user interface most commonly used until the advent of the Hermosa theme in version 8.2.

This theme supports full functionality of the developer portal, including Administrator tasks.

In Default theme:

  • Links for APIs, Apps, Groups, Forum, Organizations, and Administration are always visible in the top menu bar (Organizations and Administration are visible only if the user is logged in and has Business Admin or Site Admin rights).
