What is the purpose of an API legal agreement?

When you request access to an API, you might be required to accept a legal agreement such as an end-user license agreement, developer agreement, or service legal agreement. Legal agreements typically pertain to API usage requirements, limitations, liability, and service level expectations. The API provider is responsible for providing and enforcing the legal agreement.

How do I access an API legal agreement?

To access an API legal agreement:

  1. Go to APIs > My APIs > choose API > Agreements. The Agreements page for the selected API displays.
  2. If the API includes a single legal agreement the document will load when you access the Agreements page.
  3. If the API includes multiple legal agreements they will be presented in a series of hyperlinks. Click the hyperlink of the legal agreement you would like to view.
  4. Click the browser back button to go to the previous page.
