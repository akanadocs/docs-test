How do notifications work?

Notifications are displayed at the top of the Dashboard.

  • You can view notifications by clicking the number indicator next to the Dashboard label.
  • Each notification is presented in a summary view.
  • Five notification entries are displayed by default, and the bottom of the notification window displays the total number of notifications.
  • All notifications can be viewed by clicking the View all # notifications link. If you expand the notifications list, you can reselect the number indicator to collapse the display to the default number of notification entries.
  • You can view notification detail by clicking on the notification title. The Close button (x) removes the notification from the listing.

How do I obtain a consolidated view of activity about APIs that I am interested in?

The platform allows you to monitor activity about an API and view the API’s Forum items in your own Dashboard along with the Forum items from your app's Forum and any other APIs you are monitoring. This enables you to customize your Dashboard to give you a single view of all the news that you are most interested in. There are several options you can use in the platform to keep up-to-date with discussions and tickets for an API.

You monitor API activity when you:

  • Select Follow on the API Forum or API Details page. Notifications pertaining to the API display in your Dashboard and API Forum. See How do I follow an API?
  • Add an API to your app using the Access function in API Details page. Notifications pertaining to the API display in your Dashboard and App Forum.
  • Become a member of an app team. Notifications pertaining to APIs that are licensed with the app display in your Dashboard and App Forum.

How do I follow an API?

The API Details and API Forum include a Follow function that allows you to add the API newsfeed for the selected API to your Dashboard.

To add the API newsfeed to your Dashboard:
  1. Navigate to My APIs > API Name > Overview or My APIs > API Name > Forum.
  2. Click Follow. Newsfeed activity associated with the current API will now be added to your Dashboard. The option changes to Unfollow.
  3. See What information is displayed on my Dashboard? for a list of supported newsfeed types for the API Forum.

How do I unfollow an API?

If you have previously used the Follow function and would like to remove the API newsfeed for the selected API from your Dashboard, you can use the Unfollow function on the API Details or API Forum page.

To remove the API newsfeed from your Dashboard:
  1. Navigate to My APIs > API Name > Overview or My APIs > API Name > Forum.
  2. Click Unfollow. Newsfeed activity associated with the current API will be removed from your Dashboard. The function name changes to Unfollow.

How do I follow an app?

Two pages for the app, the Details page and Forum page, include a Follow function that allows you to follow the app. This adds information related to the app to your Dashboard; it also shows up in your "apps I'm following" list. This is particularly useful if there are a lot of apps.

To add the app newsfeed to your Dashboard:
  1. Navigate to My Apps > App > Overview or My Apps > App > Forum.
  2. Click Follow. Newsfeed activity associated with the current app will now be added to your Dashboard.
  3. For a list of supported newsfeed types for the App Forum, see What information is displayed on my Dashboard?

How do I unfollow an app?

If you'd chosen to follow an app and have changed your mind, you can choose Unfollow on the app's Details page or Forum page.

To remove the app newsfeed from your Dashboard:
  1. Navigate to My Apps > App or My Apps > App > Forum.
  2. Click Unfollow. Newsfeed activity associated with the current app will be removed from your Dashboard.
