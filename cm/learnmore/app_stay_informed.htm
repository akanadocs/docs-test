How do notifications work?

Notifications are displayed on your Notifications page, in your Action Dashboard, and in the Forum or each resource such as an app.

App notifications are issued to keep you informed about events relating to apps that you're interested in, such as:

  • Someone is invited to the app team, joins the team, or is removed from the team.
  • A contract with an API is begun or ended.
  • An app version is added or deleted.

For more information, see:

  • Preferences and Notification Settings
  • Action Dashboard
  • Forums

How do I obtain a consolidated view of activity about APIs that I am interested in?

The platform allows you to monitor activity about an API and view the API’s Forum items in your own Action Dashboard along with the Forum items from your app's Forum and any other APIs you are monitoring. This enables you to customize your Action Dashboard to give you a single view of all the news that you are most interested in. There are several options you can use in the platform to keep up-to-date with discussions and tickets for an API.

You monitor API activity when you:

  • Select Follow on the API Forum or API Details page. Notifications about the API are displayed in your Action Dashboard and in the API Forum. See How do I follow an API?
  • Add an API to your app using the Access function in API Details page. Notifications about the API are displayed in your Action Dashboard and in the App Forum.
  • Become a member of an app team. Notifications about APIs that the app is connected to are displayed in your Action Dashboard and in the App Forum.

You can also add APIs to your My Dashboard. See Monitoring APIs with the My Dashboard Feature.

How do I follow an API?

The API Overview page and API Forum both include a Follow button. When you follow an API, Alerts, Requests, and Tickets relating to the API are also displayed in your Action Dashboard.

To add API Forum entries to your Action Dashboard

  1. Go to one of these:
    •  APIs > My APIs > choose API > Overview
    •  APIs > My APIs > choose API > Forum
  2. Click Follow.

How do I unfollow an API?

If you have previously used the Follow function, but no longer want to follow the API, you can unfollow. Forum entries for the API will no longer be displayed in your Action Dashboard.

To remove API forum entries from your Action Dashboard

  1. Go to one of these:
    •  APIs > My APIs > choose API > Overview
    •  APIs > My APIs > choose API > Forum
  2. Click Unfollow.

How do I follow an app?

Two pages for the app, the Details page and Forum page, include a Follow function that allows you to follow the app. This adds information related to the app to your Action Dashboard; it also shows up in your "apps I'm following" list. This is particularly useful if there are a lot of apps.

To follow an app

  1. Go to one of these:
    • Apps > All Apps > choose app > Details
    • Apps > All Apps > choose app > Forum
  2. Click Follow.

Back to top

How do I unfollow an app?

If you'd chosen to follow an app and have changed your mind, you can choose Unfollow on the app's Details page or Forum page.

To remove app forum entries from your Action Dashboard

  1. Go to one of these:
    • Apps > All Apps > choose app > Details
    • Apps > All Apps > choose app > Forum
  2. Click Unfollow.
