Hermosa Theme

Valid in version: 2019.1.0 and later

The top navigation for Hermosa theme was redesigned in 2019.1.0. The top menu, initial vies, is shown below.

Hermosa theme supports full functionality of the developer portal, including Administrator tasks.

In the Hermosa theme:

  • Links for Dashboard, APIs and Apps are always visible in the top menu bar.
  • Groups, Organizations, Admin, and Help links are available via the More menu.
