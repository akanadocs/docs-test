Hermosa Theme

Hermosa theme, new in version 8.2, includes features such as an expanding/collapsing sidebar to offer more space for the visual display of the developer portal.

This theme supports full functionality of the developer portal, including Administrator tasks.

In the Hermosa theme:

  • Links for APIs and Apps are always visible in the top menu bar (top left in the above illustration).
  • Groups, Organizations, Admin, and Help links are available via the Advanced Options menu (vertical ellipsis, sometimes called a kebab menu) as shown above.
