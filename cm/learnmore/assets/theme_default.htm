Default Theme

Default theme is the version of the user interface most commonly used until the advent of the Hermosa theme in version 8.2.

This theme supports full functionality of the developer portal, including Administrator tasks.

In Default theme:

  • Links for APIs, Apps, Groups, and Administration are always visible in the top menu bar (Administration is visible only if the user is logged in and has Administrator rights).
