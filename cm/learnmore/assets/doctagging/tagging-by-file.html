api-admin.html

As public-scope-1 is a public scope, everyone will see the document content and will not get a 403. This includes:

  • SiteAdmin
  • BusinessAdmin
  • API Admin
  • AppAdmin
  • Registered User 01
  • Registered User 02
  • Registered User 03
  • Anonymous User

  • SiteAdmin
  • BusinessAdmin
  • API Admin
  • AppAdmin
  • Registered User 01
  • Registered User 02
  • Registered User 03

two-public-scopes.html

As private-scope-1 is a private scope, only a user who has access to private-scope-1 will see the document content and will not get a 403. This includes:

  • BusinessAdmin
  • API Admin
  • Registered User 01 (assigned to public-private-group-1 which has public-private-license-1)
  • Registered User 02 (assigned to private-group-2 which has private-license-1)

  • BusinessAdmin
  • API Admin

  • Registered User 02 (assigned to private-group-2 which has private-license-1)
