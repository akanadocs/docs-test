Dev Ops Theme

The Dev Ops theme is the Lifecycle Coordinator user interface.

Admin User: Lifecycle Coordinator opens at the Promotion Requests page, as shown below. The More menu is also shown below.

The drop-down arrow to the right of the user avatar gives access to the Profile menu (Details page only, view only) and log out, as shown below.

Note: For information about the user profile, see Account Management

 

Non-Admin user: If the user logging in does not have Admin permissions, the Admin option is not available on the More menu, as shown below.

For more information, see Managing Lifecycle Coordinator (information for Site Admins).
