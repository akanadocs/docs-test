Business Admin Content Overview

For Business Administrators: learn about how to invite others to be Business Admins and how to define licenses and scopes that comprise API Contracts used to restrict visibility and monitor service levels of API operations for apps that are connected to an API.

Business Admin Responsibilities

Overview of the Business Administrator's capabilities and duties.

Organizations

Manage organizations on the platform including adding, modifying, viewing, or deleting organizations and managing business-related tasks such as settings and workflows.

Export/Import

Export and import platform information.

Business Admins

Manage Business Administrators for the platform.

Business APIs

Manage APIs associated with an organization on the API platform.

Business Apps

Manage apps associated with an organization on the API platform.

Business Processes

Manage processes associated with an organization on the API platform.

Business Scripts

Manage scripts associated with an organization on the API platform.

Deployment Zones

Manage deployment zones for the platform.

Domains

Configure domains that can be used for login or as authentication domains for APIs defined in the platform.

OpenID Connect Support

Configure an OpenID Connect Relying Party domain that you can use for platform login or to support an OAuth Provider domain using OpenID Connect for authentication.

Forum Moderation

Moderate user-generated content relating to platform apps and APIs, such as discussions, reviews, and comments.

Managing Gateways

Manage gateways on the platform, including adding, viewing, modifying, and deleting gateways.

Licenses

Add/manage license definitions and assign to an API.

Scopes

Configure and manage scopes and licenses used to define API access contracts.
