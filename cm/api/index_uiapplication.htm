UI Application Service: Overview

The UI Application service provides operations that allow you to upload and manage platform resources such as platform CSS, customizable platform content pages, and platform help files.

URL: {protocol}/{hostname}/ui/apps/

Action Endpoint Description
get path-based resource GET /ui/apps/{application}/{version}/{path:resources/.*} Returns the platform resource specified in the path.
get script version key GET /ui/apps/{application}/scriptversionkey Returns the current script version key value.
