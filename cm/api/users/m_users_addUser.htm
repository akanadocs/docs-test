POST /api/users

Adds a new user to the platform.

Authorization Roles/Permissions: Site Admin only.

HTTP Method

POST

URL

https://<hostname>/api/users

Sample Request

The example below shows a request to add a user.

Request URL

https://<hostname>/api/users

Sample request headers

Accept: text/plain
X-Csrf-Token_<tenant>: <TokenID>

Sample request body

{
  "FirstName":"Jonathan",
  "LastName":"Swift",
  "Email":"jswift@acmepayments.com",
  "Password":"Jonathan23"
}

Response

If successful, this operation returns HTTP status code 200, with the UserID of the new user in the response message.
