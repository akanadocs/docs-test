Tenant Administration Service: Overview

The Tenant Administation service includes operations relating to managing settings for a platform tenant.

URL: https://{hostname}/api/tenants

Action Endpoint Description
get tenant GET /api/tenants/{TenantFedMemberID} Returns information about a tenant on the API Platform.
get tenant country codes GET /api/tenants/{TenantFedMemberID}/countrycodes Returns a list of telephone country codes for the tenant.
get tenant themes GET /api/tenants/{TenantFedMemberID}/themes Returns information about the themes set up for the specified tenant.
import package POST /api/tenants/{TenantFedMemberID}/packages Imports information that was previously exported from a platform version.
save tenant country codes PUT /api/tenants/{TenantFedMemberID}/countrycodes Adds or updates the list of telephone country codes that will be supported for telephone numbers in user profiles.
update tenant PUT /api/tenants/{TenantFedMemberID} Updates information about the specified platform tenant.
