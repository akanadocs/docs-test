GET /api/apis/versions/{ApiVersionID}/licensereport

Returns the report definition relating to licenses for the specified API version, for the specified ReportType.

Authorization Roles/Permissions: Must be a user with Monitor permission for the API; an API Admin, Business Admin, or App team member for an app that has a contract with the API.

Request Parameters

Parameter Parm Type Data Type Required Description
ApiVersionID Path string Required The unique ID for a specific API version.
LicenseID Query string Optional The unique ID for a specific license.
AppVersionID Query string Optional The unique ID for a specific app version.
MetricName Query string Optional The name for a specific metric.
ReportType Query string Required The type of metric information used in the license report. For valid values, see Report Type: License Report.
Environment Query string Optional The environment (Sandbox or Production). If not specified, defaults to Production.
StartDate Query string Optional The beginning point for the date range of metric information requested. If StartDate is not specified, the operation uses EndDate minus Duration. If both StartDate and EndDate are unspecified, the operation returns information for the Duration value, and if Duration is also unspecified it returns default Duration value of 2 hours.
EndDate Query string Optional The ending point for the date range of metric information requested. If not specified, defaults to the current date/time of the API call.
TimeZone Query string Optional The time zone for the information. If a time zone isn't specified, the information is returned in GMT. For more information, refer to Working with Time Zones and Time Intervals.
Duration Query string Optional The duration for the metric information. If not specified, defaults to one day (1d). The StartDate, EndDate, and Duration fields work together to define the information to be returned.
