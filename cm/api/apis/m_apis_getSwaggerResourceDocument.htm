GET /api/apis/versions/{APIVersionID}/swagger

Returns the automatically-generated Swagger resource document for the specified API version, showing the path to each resource.This operation automatically generates an inventory of all operations for the specified API version, and stores it in a Swagger file of the format specified in the parameters.

Additional information:

  • For more information about using Swagger for your API documentation on the platform, refer to Managing API Documentation with Swagger.
  • For general information about Swagger, refer to http://developers.helloreverb.com/swagger.

Authorization Roles/Permissions:   Anyone can use this operation; login is not required.
