DELETE apps/{AppID}

Deletes an app (all versions).

Note: If you want to completely remove an app, use DELETE /api/apps/{AppID}. If you want to delete the only existing app version, but keep the app, first add a new version, using POST apps/{AppID}/versions, and then delete the old version using DELETE /api/apps/versions/{RuntimeID}. An app cannot exist in the Enterprise API Platform without at least one valid version.

Authorization Roles/Permissions: App team member, Business Administrator

Sample Request

The example below shows a request to delete a specific app.

Sample request URL

https://<hostname>/api/apps/app10035.acmepaymentscorp

Sample request headers

DELETE /api/apps/app10035.acmepaymentscorp HTTP/1.1
Content-Type: text/plain
Accept: text/plain
X-Csrf-Token_{tenant}: <TokenID>

Sample request body

None.
