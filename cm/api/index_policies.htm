Policies Service: Overview

The Policies service supports managing the policies that govern API operation in the platform.

URL: https://{hostname}/api/policies

Action Endpoint Description
Get Policies GET /api/policies Returns a list of supported policies, such as Service Level, Management, or Compliance policies.
