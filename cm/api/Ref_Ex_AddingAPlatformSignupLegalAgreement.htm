Example: Adding a Platform Signup Legal Agreement

This example gives you the basic steps for adding a new platform signup legal agreement in Enterprise API Platform by using the API rather than the user interface.

This exercise assumes that you already have a valid login for Enterprise API Platform and are a Site Admin.

Before starting, you will need:

  • A valid UserID and password for platform access.
  • The legal agreement itself.

For more information about legal agreements, refer to Managing Legal Agreements on the Platform.

Essentially, you'll need to complete these activities, in sequence:

  1. First, make sure there are no existing platform signup legal agreements. The platform allows only one. If there are any already existing, you will have to run a database script for deactivation. For instructions, refer to the Community Manager Site Admin help. If it is a new installation, you can skip this step.
  2. Log in using the authorized user profile you've already set up. Operation: POST /api/login.
  3. Upload the platform legal agreement using the Content API. Operation: POST /content/upload.
  4. Define the new agreement as the platform's signup legal agreement. Operation: POST /api/legals.
  5. When done, remember to log out using the POST /api/login/logout operation.
