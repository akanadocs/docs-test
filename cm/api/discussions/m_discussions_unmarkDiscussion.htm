DELETE /api/discussions/{DiscussionID}/marks/{UserID}

Removes the user's mark from a discussion that the user has previously marked.

Authorization Roles/Permissions: Registered site user.

URL

Sample Request

The example below shows a request to remove a mark from the specified discussion.

Request URL

https://<hostname>/api/discussions/discussion25800.acmepaymentscorp/marks/user12262.acmepaymentscorp

Sample request headers

DELETE /api/discussions/discussion25800.acmepaymentscorp/marks/user12262.acmepaymentscorp HTTP/1.1
Host: <hostname>
Accept: */*
Content-Type: application/x-www-form-urlencoded; charset=UTF-8
X-Csrf-Token_{tenant}: <TokenID>

Sample request body

None.

Response

If successful, this operation returns HTTP status code 200, with the updated count of the number of marks on the specified discussion.
