GET /api/discussions/{DiscussionID}/details

Returns information about a discussion, including details such as workflow steps that are valid for the discussion and, possibly, comments and target resources relating to the discussion.

Authorization Roles/Permissions: This operation applies to all users. Results are only returned if the user has visibility of the specific discussion.
