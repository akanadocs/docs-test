Dropbox Service: Overview

The Dropbox service provides operations for uploading images to the Enterprise API Platform, to support operations that create or modify resources that might include an image, such as user profiles.

The Dropbox provides a temporary storage area. If you want to upload an image to the platform, you can first upload it to the Dropbox. The Dropbox returns a token ID or a picture ID, and you can then use that ID to retrieve the image using the GET /api/dropbox/pictures/{pictureId} operation, or as part of methods such as POST apps or POST /api/apis. The method takes the picture from the dropbox; it is then stored on the server.

The Dropbox is only temporary. Dropbox items are stored for a short time and then deleted.

URL: https://<hostname>/api/dropbox

Action Endpoint Description
add picture POST /api/dropbox/pictures Adds an image for a resource such as a user profile.
crop picture PUT /api/dropbox/pictures/{pictureId} Crops a picture associated with a resource such as a user profile, app, or API, and updates the database.
get picture GET /api/dropbox/pictures/{pictureId} Returns a picture that was uploaded to the Dropbox for a resource.
read WSDL ZIP POST /api/dropbox/wsdls Accepts a ZIP file containing a WSDL and responds with an list of operations.
