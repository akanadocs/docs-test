APIInfos Object

Contains information about one or more APIs that were defined using a Swagger 2.0 description document.

This object is used by the following operations:

  • GET /api/apis (with specific headers)

The properties included in the APIInfos object are listed below.

Property Type Description
APIInfo[ ] APIInfo Contains information about an API that was defined using a Swagger 2.0 description document.
