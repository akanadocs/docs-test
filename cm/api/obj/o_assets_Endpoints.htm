Endpoints Object

Contains information about endpoints associated with a service.

This object is used by the following operations:

  • Operations that reference the ServiceAsset object
  • Operations that reference the ServiceReference object

The properties included in the Endpoints object are listed below.

Property Type Description
Endpoint Endpoint Includes information about an endpoint for a specific asset.
