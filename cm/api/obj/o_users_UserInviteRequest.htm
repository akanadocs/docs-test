UserInviteRequest Object

Contains information about an invitation to a user group such as an app team or private API group.

This object is used by the following operations:

  • POST /api/users/{UserID}/invitations

The properties included in the UserInviteRequest object are listed below.

Property Type Description
EmailAddresses string[ ] One or more email addresses of individuals, separated by commas. Invitees do not have to be registered users. Optional.
Message string A message to be included with the invitation. Optional.
