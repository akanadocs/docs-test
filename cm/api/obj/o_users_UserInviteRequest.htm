UserInviteRequest Object

Contains information about an invitation to a user group such as an app team or API Scope Group.

This object is used by the following operations:

  • POST /api/users/invitations

The properties included in the UserInviteRequest object are listed below.

Property Type Description
EmailAddresses string[ ] One or more email addresses of individuals that are being invited, separated by commas. Optional.
Message string A message to be included with the invitation. Optional.
