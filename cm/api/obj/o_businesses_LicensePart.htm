LicensePart Object

Contains information about a license part.

This object is used by the following operations:

  • Any operations that use the LicenseParts object

The properties included in the LicensePart object are listed below.

Property Type Description
Name string The name of the license part.
ResourceID[ ] string

The unique ID for a specific resource; for example, the ScopeID or the APIVersionID.

One or more.
PolicyKey[ ] string

The unique key for a specific policy associated with the license.

One or more.
