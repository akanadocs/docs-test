APIBindings Object

Contains information about one or more API bindings for the implementation.

This object is used by the following operations:

  • Operations that reference the APIImplementation object
  • Operations that reference the TargetAPI object

The properties included in the APIBindings object are listed below.

Property Type Description
APIBinding APIBinding[ ] Contains information about an API binding for the implementation.
