ContractScope Object

Contains information about the scope for an app/API contract.

This object is used by the following operations:

  • Operations that reference the Contract object

The properties included in the ContractScope object are listed below.

Property Type Description
Restricted boolean Indicates whether the contract is restricted.
LicenseID string[ ]

The unique ID for a specific license.

One or more licenses applicable to the contract.
