APIVersions Object

Contains information about one or more API versions.

This object is used by the following operations:

  • GET /api/apis/versions

The properties included in the APIVersions object are listed below.

Property Type Description
APIVersion APIVersion[ ]

Includes information about an API version.

One or more.
