APIVersionOAuthDetails Object

Includes OAuth information about an API version, such as OAuth version and settings.

This object is used by the following operations:

  • PUT /api/apis/versions/{APIVersionID}/oauthdetails (with specific header) (in the request) (2019.0.0 and later)
  • PUT /api/apis/versions/{APIVersionID}/oauthdetails (in the request) (Legacy operation in 2019.0.0; instead, use PUT /api/apis/versions/{APIVersionID}/oauthdetails)

The properties included in the APIVersionOAuthDetails object are listed below.

Property Type Required Description
APIVersionID string Required The unique ID for a specific API version.
TargetAPI EndpointsOAuthDetails Optional Includes OAuth information for an API's Target API endpoints, if applicable.
ProxyAPI EndpointsOAuthDetails Optional Includes OAuth information for an API's Proxy API endpoints, if applicable.
OperationsOAuthDetails OperationsOAuthDetails[ ] Optional Includes OAuth information for individual operations within the API, if applicable.
APIVersionOAuthResourceMapping APIVersionOAuthResourceMapping Optional Contains information about one or more resources (grants) applicable to an API-wide resource.
