APIImplementationInfos Object

Contains information about one or more implementations for an API version.

This object is used by the following operations:

  • GET /api/apis/versions/implementations

The properties included in the APIImplementationInfos object are listed below.

Property Type Description
APIImplementationInfo APIImplementationInfo[ ]

Contains information about an API implementation.

One or more.
