WfActionDefinition Object

Contains information about a specific workflow action.

The properties included in the WfActionDefinition object are listed below.

Property Type Description
ActionName string The name of the workflow action.
StateName string The name of the workflow state.
StepID integer The numeric ID for the workflow action.
