APIInterfaces Object

Contains information about one or more API interfaces.

This object is used by the following operations:

  • Operations that reference the APIImplementation object
  • Operations that reference the TargetAPI object

The properties included in the APIInterfaces object are listed below.

Property Type Description
APIInterface APIInterface[ ]

Contains information about an API interface.

One or more.
