LicensePartsDetails Object

Contains information about one or more license parts.

This object is used by the following operations:

  • Any operations that use the LicenseDetails object

The properties included in the LicensePartsDetails object are listed below.

Property Type Description
LicensePartDetails LicensePartDetails Contains details about a license part.
