APIVersionInfo Object

Contains API version info for an API defined using a Swagger 2.0 description document.

This object is used by the following operations:

  • PUT /api/apis/versions/{ApiVersionID}/info (in the response)
  • Operations that reference the addAPIRequest object

The properties included in the APIVersionInfo object are listed below.

Property Type Description
APIVersionID string

The unique ID for a specific API version.

Used only in responses.
Name string The name of the API.
Description string The description of the API.
APIInfo APIInfo Contains information about an API that was defined using a Swagger 2.0 description document.
Tags string[ ]

One or more user-defined tags assigned to the API, separated by commas. Used only in response messages.

Note: You can also assign specific tags to each API version. In the user interface, only the tags assigned to at the API version level are displayed.
Visibility string The visibility setting for the API; either Public, which is visible to all users, or Limited, which is visible only to API Administrators and API Scope Group members.
