UserRoles Object

Contains information about a user's roles in relation to a resource.

This object is used by the following operations:

  • GET /api/users/{UserID}/roles?ResourceID={ResourceID}

The properties included in the UserRoles object are listed below. All values are optional.

Property Type Description
RoleName string[ ] An array of one or more roles assigned to the user.
