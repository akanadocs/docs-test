SchemaImports Object

Contains information about a model schema import.

This object is used by the following operations:

  • Operations that reference the SchemaDocumentReference object

The properties included in the SchemaImports object are listed below.

Property Type Description
Name string The name of the schema.
AddToLibrary boolean Indicates whether the schema should be added to the library on the platform.
SchemaImports SchemaImports Contains information about one or more model object schemas within an uploaded model object schema.
