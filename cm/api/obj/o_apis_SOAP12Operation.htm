SOAP12Operation Object

Contains information about a SOAP 1.2 operation.

This object is used by the following operations:

  • Any operations that use the BindingOperation object

The properties included in the SOAP12Operation object are listed below.

Property Type Description
SOAPAction string Contains information about a SOAP 1.2 SOAPAction header associated with a SOAP 1.2 request message for a specific operation.
