BusinessResources Object

Contains information about scopes defined for a business organization.

This object is used by the following operations:

  • GET /api/businesses/{BusinessID}/resourcehierarchy (in the response)

The properties included in the BusinessResources object are listed below.

Property Type Description
Resource Resource[ ]

Contains information about a resource, such as a scope, defined for a business organization.

One or more.
