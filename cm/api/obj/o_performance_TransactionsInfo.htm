TransactionsInfo Object

Contains details about one or more transactions. This object is used by two services: apps and apis.

This object is used by the following operations:

  • GET /api/apps/versions/{AppVersionID}/txlogs
  • GET /api/apis/versions/{APIVersionID}/environments/{Env}/operations/{OpName}/txlogs
  • GET /api/apps/versions/{AppVersionID}/txlogs

The properties included in the TransactionsInfo object are listed below.

Property Type Description
TransactionInfo TransactionInfo[ ] Contains information about one or more transactions for an app or API, with details for each transaction.
