DZoneEnvImplAssociations Object

Contains information about implementation/environment associations for a deployment zone.

This object is used by the following operations:

  • PUT /api/deploymentzones/{DZoneID}/associations (in request and response)
  • GET /api/deploymentzones/{DZoneID}/associations (in response)

The properties included in the DZoneEnvImplAssociations object are listed below.

Property Type Description
DZoneEnvImplAssociation DZoneEnvImplAssociation[ ] Contains information about implementation/environment associations for a deployment zone.

One or more.
