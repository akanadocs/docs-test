SummaryType Object

Contains information about a type of metric summary.

This object is used by the following operations:

  • Operations that reference the Graph object

The properties included in the SummaryType object are listed below.

Property Type Description
Metric Metric[ ] Contains information about an array of metrics for the summary interval.
StartTime dateTime The start time for the information being reported.
EndTime dateTime The end time for the information being reported.
Duration duration The Duration for the metric summary interval in the format PxM where M is minutes and x is the number of minutes.
