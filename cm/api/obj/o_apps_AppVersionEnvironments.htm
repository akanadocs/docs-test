AppVersionEnvironments Object

Contains information about the environments available for the app version to request a contract with the API version.

This object is used by the following operations:

  • GET /api/apps/versions/{RuntimeID}/contracts/environments
  • Any operations that use the AppVersionEnvironment object

The properties included in the AppVersionEnvironments object are listed below.

Property Type Description
AppVersionEnvironment AppVersionEnvironment Contains information about an environment available for an app version/API version contract.
