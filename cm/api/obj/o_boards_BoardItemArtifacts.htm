BoardItemArtifacts Object

Contains information about the images and/or other artifacts associated with a Board item.

This object is used by the following operations:

  • Operations that reference the Discussion object
  • Operations that reference the Ticket object
  • Operations that reference the Alert object

The properties included in the BoardItemArtifacts object are listed below.

Property Type Description
BoardItemArtifact BoardItemArtifact Contains information about an image or other artifact associated with a Board item. Applicable only if the DescriptionMediaType for the Board item is text/markdown.
