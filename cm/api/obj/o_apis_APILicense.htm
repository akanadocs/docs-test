APILicense Object

Contains information about a specific license associated with an API version.

This object is used by the following operations:

  • Any operations that use the APILicenses object

The properties included in the APILicense object are listed below.

Property Type Description
LicenseDetails LicenseDetails Contains details about a specific license.
