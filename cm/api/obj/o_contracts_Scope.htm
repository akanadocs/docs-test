Scope Object

Contains information about a scope.

This object is used by the following operations:

  • Any operations that use the Contract object

The properties included in the Scope object are listed below.

Property Type Description
Restricted boolean Indicates whether the scope is restricted.
LicenseID string[ ]

The unique ID for a specific license.

One or more licenses applicable to the contract.
