BoardItems Object

Contains information about one or more board items.

This object is used by the following operations:

  • GET /api/boards/{BoardID}/board

The properties included in the BoardItems object are listed below.

Property Type Description
BoardItem BoardItem[ ]

Contains information about a board item.
