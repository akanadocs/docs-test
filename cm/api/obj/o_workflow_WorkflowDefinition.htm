WorkflowDefinition Object

Contains information about a workflow.

This object is used by the following operations:

  • POST /api/workflow/definitions (in response)

The properties included in the WorkflowDefinition object are listed below.

Property Type Description
Key string The unique workflow definition key for the custom workflow.
Name string The workflow name.
Description string The workflow description.
State string The current state of the workflow.
ObjectType string The type of resource the workflow is for. For valid values, see Workflow Object Types.
BusinessID string The unique ID for a specific business organization on the platform.
Updated dateTime The date/time stamp when the resource was last updated. Used only in response messages.
ObjectReferenceCount Long Object Reference count.
