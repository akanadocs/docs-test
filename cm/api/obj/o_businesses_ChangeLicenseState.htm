ChangeLicenseState Object

Contains information about the state of a business license.

This object is used by the following operations:

  • PUT /api/licenses/{LicenseID}/state

The properties included in the ChangeLicenseState object are listed below.

Property Type Description
Active Boolean Indicates whether the license is active or not.
