TenantStateChangeRequest Object

Contains information about a request to change the state of a tenant on the platform.

This object is used by the following operations:

  • PUT /platform/subscriptions/{SubscriptionID}/state

The properties included in the TenantStateChangeRequest object are listed below.

Property Type Description
TargetState string

The new state for the tenant. Valid values:

  • com.akana.tenant.state.active
  • com.akana.tenant.state.suspended
