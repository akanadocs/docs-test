AppInfos Object

Contains summary information about one or more apps.

This object is used by the following operations:

  • GET /api/apps

The properties included in the AppInfos object are listed below.

Property Type Description
AppInfoType[ ] AppInfoType

Contains summary information about an app.

One or more.
