OperationResourceDetails Object

Contains details about operations associated with an API version and the scopes assigned to them, if operation-specific scope mapping is in use for the API version.

This object is used by the following operations:

  • Any operations that reference the APIVersionResourceDetails object
  • Any operations that reference the APIVersionResources object
  • Any operations that reference the Operation object

The properties included in the OperationResourceDetails object are listed below.

Property Type Description
OperationName string The name of the operation.
Resource Resource[ ] Information about one or more resources (grants) associated with an operation.
