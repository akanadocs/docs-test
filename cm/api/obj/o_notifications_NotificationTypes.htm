NotificationTypes Object

Contains information about the types of notifications available on the platform.

This object is used by the following operations:

  • GET /api/notifications/types

The properties included in the NotificationTypes object are listed below.

Property Type Description
NotificationType[ ] string Contains information about a notification type (notification grouping) available on the platform.
