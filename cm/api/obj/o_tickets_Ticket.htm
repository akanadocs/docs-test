Ticket Object

Includes information about a ticket.

This object is used by the following operations:

  • POST /api/tickets
  • GET /api/tickets/{TicketID}
  • Operations that reference the Tickets object

The properties included in the Ticket object are listed below.

Property Type Required Description
TicketID string Optional The unique ID for a specific ticket.
APIID string Required The unique ID for a specific API.It is the ID for the API the ticket is associated with, if applicable.
AppID string Optional The unique ID for a specific app.It is the ID of the app the ticket is associated with.
Title string Optional The ticket title. Required when adding a ticket.
Description string Optional The text description of the ticket. Required when adding a ticket.
Environment string Optional The environment to which the ticket relates (Sandbox or Production). Required when adding a ticket.
State string Optional The current state of the ticket. For valid values, see Ticket State Values.
UserID string Required The unique ID for a specific user. It is the ID of the user who created the ticket.
UserName string Optional The username of the person who created the ticket.
Created dateTime Optional The date/time stamp when the resource was first created. Used only in response messages.
Label string[ ] Optional

One or more ticket labels indicating the priority of the ticket. When setting priority, specify only one value. For valid values, see Ticket Label Values.

Required when adding a ticket.
Tag string[ ] Optional One or more user-defined tags assigned to the ticket.

Comments int Optional The number of comments associated with the ticket.
Marks int Optional The number of marks on the resource.
IsMarked boolean Optional Indicates whether there are any marks on the resource or not.
Visibility string Optional The visibility setting for the API; either Public, which is visible to all users, or Limited, which is visible only to API Administrators and Private API Group members.
pubDate dateTime Optional The publication date for the discussion, formatted in Coordinated Universal Time (UTC).
CommentsPreview CommentsPreview Optional A preview of the comments added to the ticket (the last two comments plus total count).
Action Action[ ] Optional

An extensibility element for a workflow action event that possibly could be taken on a specific resource. Used to call a JavaScript action.

A list of actions available on the ticket.
AuthoringUser AuthoringUser Optional

Contains information about the author of a resource.

The author who created the ticket.
TargetResources TargetResources Optional Contains information about one or more resources that are associated with the current resource.
Image Image Optional Contains information about an avatar associated with a resource.
