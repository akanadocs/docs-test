APIVersionAutoConnectLicenseDetails Object

Contains information about auto-connect details for one or more environments, for an API version.

This object is used by the following operations:

  • PUT /api/apis/versions/{APIVersionID}/autosettings (in the response)
  • GET /api/apis/versions/{APIVersionID}/autosettings (in the response)

The properties included in the APIVersionAutoConnectLicenseDetails object are listed below.

Property Type Description
APIVersionAutoConnectLicenseDetail APIVersionAutoConnectLicenseDetail Contains information about auto-connect details for one or more environments, for an API version.
