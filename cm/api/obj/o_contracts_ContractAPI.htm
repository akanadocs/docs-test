ContractAPI Object

Contains information about the API that's part of a specific app/API contract.

This object is used by the following operations:

  • GET /api/contracts/{ContractID}/apidetails

The properties included in the ContractAPI object are listed below.

Property Type Description
API API Includes information about an API.
APIVersion APIVersion Includes information about an API version.
