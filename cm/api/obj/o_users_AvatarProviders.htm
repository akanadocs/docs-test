AvatarProviders Object

Contains information about one or more avatar providers.

This object is used by the following operations:

  • GET /api/users/{UserID}/avatarproviders

The properties included in the AvatarProviders object are listed below. All values are optional.

Property Type Description
AvatarProvider AvatarProvider[ ] Contains information about a user's avatar provider.
