BindingOperations Object

Contains information about one or more operations associated with a binding.

This object is used by the following operations:

  • Any operations that use the APIBinding object

The properties included in the BindingOperations object are listed below.

Property Type Description
BindingOperation BindingOperation Contains information about an operation associated with a binding.
