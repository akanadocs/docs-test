CountryCode Object

Contains information about a telephone country code supported on the platform.

This object is used by the following operations:

  • Operations that reference the CountryCodes object

The properties included in the CountryCodes object are listed below.

Property Type Description
id string The ID for the telephone country code; for example, 1 for the United States and Canada, or 41 for Switzerland.
Name string The name of the country; for example, United States and Canada or Switzerland.
