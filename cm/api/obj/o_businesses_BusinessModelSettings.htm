BusinessModelSettings Object

Contains information about default settings to apply to all business models within the business.

This object is used by the following operations:

  • PUT /api/businesses/{BusinessID}/modelsettings (in request and response)
  • GET /api/businesses/{BusinessID}/modelsettings (in response)

The properties included in the BusinessModelSettings object are listed below.

Property Type Description
ModelVersionWorkflowDefKey string The unique workflow definition key for the workflow definition file for model objects in the business organization.
