UserStatus Object

Contains statistical information about the status of a registered user.

This object is used by the following operations:

  • GET /api/users/{UserID}/status

The properties included in the UserStatus object are listed below.

Property Type Description
APIs int The number of APIs the user is following.
NewFollowRequests int The number of open follow requests; that is, the request was made and has not yet been accepted or rejected.
Followings int The number of resources the user is following.
Discussions int The number of discussions the user started.
Tickets int The number of tickets the user has opened.
