BusinessUserSettings Object

Default settings to apply to all users within the business, unless specified otherwise at a lower level.

This object is used by the following operations:

  • Any operations that use the Resource object
  • Any operations that use the BusinessSettings object
  • Any operations that use the Business object

The properties included in the BusinessUserSettings object are listed below.

Property Type Description
NewsUpdateNotification string Indicates whether news update notifications are supported for the business.
