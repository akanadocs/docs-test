APIVersionConnectionEnvironment Object

Contains information about a connection environment valid for an API version.

This object is used by the following operations:

  • Operations that reference the APIVersionConnectionEnvironments object

The properties included in the APIVersionConnectionEnvironment object are listed below.

Property Type Description
EnvironmentID string ID for the environment. By default, this is the same as the Environment field below (Sandbox or Production).
Environment string Environment (Production or Sandbox).
