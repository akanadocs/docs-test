Metric Object

Contains information about a metric.

This object is used by the following operations:

  • Operations that reference the IntervalType object
  • Operations that reference the SummaryType object

The properties included in the Metric object are listed below.

Property Type Required Description
Name string Required The name for the metric.
Key string Optional The metric key. This value corresponds to the column name in the metrics export file, if the information is exported; the information type. For example, avgResponseTime, totalCount, or faultCount. For valid values, see Metric Key Values.
Value integer Required The value for the metric.
