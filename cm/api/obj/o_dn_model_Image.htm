Image Object

Contains information about an avatar associated with a resource.

Note: This information was previously defined as part of an RSS extension, for operations that return an RSS feed. It is now defined as a separate object.

This object is used by the following operations:

  • Any operations that use the TargetResource object
  • Any operations that use the AuthoringUser object

The properties included in the Image object are listed below.

Property Type Description
URL anyURI The URL of the image file.
Title string A title of the image; used in the ALT attribute when the image is rendered in HTML.
Description string Text that is included in the <title> attribute of the link formed around the image in the HTML rendering.
