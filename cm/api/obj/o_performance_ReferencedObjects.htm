ReferencedObjects Object

Contains information about objects referenced in a license report, such as an app, API, or contract.

This object is used by the following operations:

  • Operations that reference the Report object

The properties included in the ReferencedObjects object are listed below.

Property Type Description
License License[ ] Contains information about an API license defined for a business.
APIVersion APIVersion Includes information about an API version.
API API Contains information about an API.
ApplicationVersion ApplicationVersion[ ]

Contains information about a version of an app.

One or more.
Application Application[ ]

Contains information about an app.

One or more.
Policies Policies Information about one or more policies applicable to the report data.
Contract Contract Contains information about a contract between an app and an API.
