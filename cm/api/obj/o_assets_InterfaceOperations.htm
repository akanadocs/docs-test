InterfaceOperations Object

Contains information about an operation that is part of the API interface for an asset.

This object is used by the following operations:

  • Operations that reference the ServiceInterface object

The properties included in the InterfaceOperations object are listed below.

Property Type Description
InterfaceOperation InterfaceOperation[ ]

Contains information about an operation that is part of the API interface for an asset.

One or more.
