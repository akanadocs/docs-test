ImplEnvAssociations Object

Contains information about one or more implementation/environment associations.

This object is used by the following operations:

  • GET /api/businesses/{BusinessID}/implenvassociations

The properties included in the ImplEnvAssociations object are listed below.

Property Type Description
ImplEnvAssociation ImplEnvAssociation[ ]

Contains information about the association between an implementation and an environment.

One or more.
