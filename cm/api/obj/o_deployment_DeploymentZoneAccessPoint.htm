DeploymentZoneAccessPoint Object

Contains information about an access point for a deployment zone.

This object is used by the following operations:

  • Operations that reference the DeploymentZoneInfo object

The properties included in the DeploymentZoneAccessPoint object are listed below.

Property Type Description
Protocol string The protocol for the deployment zone access point.
Name string A designated name for the deployment zone access point.
Address string The address for the deployment zone access point.
