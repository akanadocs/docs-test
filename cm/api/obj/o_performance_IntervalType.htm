IntervalType Object

Contains information about an interval type being reported for a metric test.

This object is used by the following operations:

  • Operations that reference the Graph object
  • Operations that reference the ApiMetrics object

The properties included in the IntervalType object are listed below.

Property Type Required Description
Metric Metric[ ] Required Information about one or more metric performance tests.
GeoLocationMetric GeoLocationMetric[ ] Required Information about the location. One or more.
StartTime dateTime Required The start time for the information being reported.
