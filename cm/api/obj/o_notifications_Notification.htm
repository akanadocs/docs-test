Notification Object

Contains information about a notification on the developer portal.

The properties included in the Notification object are listed below.

Property Type Description
NotificationID String The unique ID for a specific instance of a notification on a specific user's Dashboard.
Subject String The subject value for the notification.
NotificationType String The notification type. For valid values, see Notification Type values.
Content String The content of the notification.
Created dateTime The date/time stamp when the resource was first created. Used only in response messages.
EventID String Unique ID for the event that triggered the notification.
