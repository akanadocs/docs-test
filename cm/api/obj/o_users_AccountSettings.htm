AccountSettings Object

Contains information about the user's account settings, including basic information and the user's choices with regard to notification emails.

This object is used by the following operations:

  • PUT /api/users/{UserID}/settings
  • GET /api/users/{UserID}/settings

The properties included in the AccountSettings object are listed below.

Property Type Description
UserID string The unique ID for a specific user.Required for request objects.
Password string The user's password.
Email string The user's email address. Required for request objects.
Phone string Phone number for the user.
OptedNotification string[ ] An array of notifications groups for which the specified user will receive email notifications. Any notifications groups the user has opted out of do not appear in the array. For a list of valid values, see Notification Group Values. Required for request objects.
