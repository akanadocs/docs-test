ResolveLoginPendingTaskRequest Object

Contains information about a custom login task that is pending for the current user.

This object is used by the following operations:

  • POST /api/login/pendingtasks

The properties included in the ResolveLoginPendingTaskRequest object are listed below.

Property Type Description
taskID string The value identifying the pending task.
parameters[ ] string The parameters defined for the task.
