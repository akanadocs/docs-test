TargetResources Object

Contains information about one or more resources associated with the current resource. Used only in response messages.

Note: This information was previously defined as part of an RSS extension, for operations that return an RSS feed. It is now defined as a separate object.

This object is used by the following operations:

  • Operations that reference the Contract object
  • Operations that reference the Discussion object
  • Operations that reference the DiscussionDetails object
  • Operations that reference the Ticket object
  • Operations that reference the BoardItemDetails object
  • Operations that reference the AlertDetails object
  • Operations that reference the AgreementDocument object

The properties included in the TargetResources object are listed below.

Property Type Description
TargetResource TargetResource[ ]

Contains information about a resource that is associated with the current resource.

One or more.
