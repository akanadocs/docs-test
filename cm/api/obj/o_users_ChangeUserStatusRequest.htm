ChangeUserStatusRequest Object

Contains information about a user's change of status, with comments.

This object is used by the following operations:

  • PUT /api/users/{UserID}/status

The properties included in the ChangeUserStatusRequest object are listed below.

Property Type Required Description
UserStatus string Optional The user's status. For a list of valid values, see User Status.
Comments string Required Comments entered by the user when performing the action.
