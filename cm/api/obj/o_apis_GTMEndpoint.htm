GTMEndpoint Object

Contains information about the endpoint for a GTM.

This object is used by the following operations:

  • Operations that reference the EndpointImplementationDetails object

The properties included in the GTMEndpoint object are listed below.

Property Type Description
EndpointKey string Unique key for the endpoint.
GTMInstanceID string Unique ID for the GTM endpoint instance.
GTMType string The type of GTM.
