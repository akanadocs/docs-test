ContractApplication Object

Contains information about the app that's part of a specific app/API contract.

This object is used by the following operations:

  • GET /api/contracts/{ContractID}/appdetails

The properties included in the ContractApplication object are listed below.

Property Type Description
Application Application Contains information about an app.
ApplicationVersion ApplicationVersion Contains information about a version of an app.
