OAuthClient71 Object

Includes information about the OAuth Client settings for the specified app. This object is the payload for legacy operation PUT /api/apps/versions/{AppVersionID}/oauthclient. The newer operation is PUT /api/apps/versions/{AppVersionID}/oauthclient (same endpoint with different payload and content-type).

This object is used by the following operations:

  • PUT /api/apps/versions/{AppVersionID}/oauthclient

The properties included in the OAuthClient object are listed below.

Property Type Description
BaseOAuthClient BaseOAuthClient Includes information about the OAuth Client settings for the specified app.
RedirectURI string

Optional. Redirect URI to which the end-user will be redirected after authentication. The OAuth Provider validates the RedirectURI in the message against the RedirectURI set up in the profile.

Note: In this object, only one RedirectURI is supported. In the newer operation (see link above), multiple redirect URIs are supported.
