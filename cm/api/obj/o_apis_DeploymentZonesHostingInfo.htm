DeploymentZonesHostingInfo Object

Contains hosting information for a specific deployment zone associated with an API implementation.

This object is used by the following operations:

  • Operations that reference the APIImplementationDeployments object

The properties included in the DeploymentZonesHostingInfo object are listed below.

Property Type Description
DeploymentZoneID string Unique ID for a specific deployment zone.
DeploymentZoneEndpoint DeploymentZoneEndpoint[ ]

Contains information about a deployment zone endpoint.

One or more.
