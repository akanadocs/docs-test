Environments Object

Contains information about one or more environments.

This object is used by the following operations:

  • GET /api/businesses/{BusinessID}/environments

The properties included in the Environments object are listed below.

Property Type Description
Environment string[ ]

The environment (implementation) (Sandbox or Production).

One or more.
