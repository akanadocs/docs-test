GeoLocationMetric Object

Contains information about geographical location being reported for a metric test.

This object is used by the following operations:

  • Operations that reference the IntervalType object

The properties included in the GeoLocationMetric object are listed below.

Note: Only one of the properties below is used.

Property Type Required Description
Location string Required A text title for the location.
Latitude double Required Information about the latitude of the location.
Longitude double Required Information about the longitude of the location.
Count integer Required The count.
