ApplicationVersions Object

Contains information about one or more app versions.

This object is used by the following operations:

  • GET /api/users/{UserID}/apps
  • any operations that use the Application object

The properties included in the ApplicationVersions object are listed below.

Property Type Description
ApplicationVersion ApplicationVersion[ ]

Contains information about a version of an app.

One or more.
