UserTopAPIs Object

Contains information about APIs that a user has chosen to follow as Top APIs.

This object is used by the following operations:

  • GET /api/users/{UserID}/topapis (in the response)

The properties included in the UserTopAPIs object are listed below.

Property Type Description
UserTopAPI UserTopAPI[ ]

Contains information about an API that a user has chosen as a Top API.

One or more, up to the maximum of 10
