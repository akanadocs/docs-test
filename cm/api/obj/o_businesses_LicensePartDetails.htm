LicensePartDetails Object

Contains details about a license part.

This object is used by the following operations:

  • Operations that reference the LicensePartsDetails object

The properties included in the LicensePartDetails object are listed below.

Property Type Description
Name string The name for the license part.
Resource Resource Contains information about a resource, such as a scope, defined for a business organization.
Policies Policies Contains information about one or more policies that are enforced for the API version, if applicable.
