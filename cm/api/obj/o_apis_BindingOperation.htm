BindingOperation Object

Contains information about an operation associated with a binding.

This object is used by the following operations:

  • zzz
  • zzz
  • Any operations that use the BindingOperations object

The properties included in the BindingOperation object are listed below.

Property Type Description
OperationName string The name of the operation.
BindingOperationKey string a unique ID field for internal use only.
WsdlOperationName string The name of the operation as it is represented in the WSDL.
SOAP11Operation SOAP11Operation Contains information about a SOAP 1.1 operation.
SOAP12Operation SOAP12Operation Contains information about a SOAP 1.2 operation.
HttpOperation HttpOperation Contains information about a REST-based operation.
