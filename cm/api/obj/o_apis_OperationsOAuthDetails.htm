OperationsOAuthDetails Object

Includes information one or more OAuth operations.

This object is used by the following operations:

  • Operations that reference the APIVersionOAuthDetails object; for example, the PUT /api/apis/versions/{APIVersionID}/oauthdetails operation

The properties included in the OperationsOAuthDetails object are listed below.

Property Type Description
OperationsOAuthDetails OperationOAuthDetails[ ] Includes information about OAuth details applicable to one or more operations valid for an API.
