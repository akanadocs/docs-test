AuthoringUser Object

Contains information about the author of a resource.

An extensibility element identifying the author's user id in the system. It is different than the standard author element which is an email address.

Note: This information was previously defined as part of an RSS extension, for operations that return an RSS feed. It is now defined as a separate object.

This object is used by the following operations:

  • Any operations that use the Ticket object

The properties included in the AuthoringUser object are listed below.

Property Type Description
UserID string The unique UserID for the author of the resource.
Name string The name of the author of the resource.
Image Image Contains information about an avatar associated with a resource.
