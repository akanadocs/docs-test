AgreementDocuments Object

Contains Information about the legal agreement documents matching the specified criteria.

This object is used by the following operations:

  • GET /api/legals?AgreementType={LegalAgreementType} (with specific headers) (iin response)

The properties included in the AgreementDocument object are listed below. Properties are optional unless otherwise specified.

Property Type Description
AgreementDocument[ ] AgreementDocument

Contains information about a platform legal agreement document.

One or more.
