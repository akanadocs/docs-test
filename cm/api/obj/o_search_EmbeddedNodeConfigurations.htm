EmbeddedNodeConfigurations Object

Contains information about the Elasticsearch embedded node configuration for all the containers in an implementation.

This object is used by the following operations:

  • GET /api/index/nodeconfig (in the response)

The properties included in the EmbeddedNodeConfigurations object are listed below.

Property Type Description
EmbeddedNodeConfiguration EmbeddedNodeConfiguration Contains information about the Elasticsearch embedded node configuration for a container.
