BoardItemAssignments Object

Contains information about resources assigned to a board.

This object is used by the following operations:

  • Operations that reference the AbstractBoardItemDetails object

The properties included in the BoardItemAssignments object are listed below.

Property Type Description
ResourceID string[ ] The unique ID for a specific resource; for example, the ScopeID or the APIVersionID.
