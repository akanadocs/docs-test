Businesses Object

Contains information about one or more businesses.

This object is used by the following operations:

  • GET /api/businesses (with vendor-specific headers)

The properties included in the Businesses object are listed below.

Property Type Description
Business Business[ ]

Contains information about a business on the platform.

One or more.
