Contracts Object

Includes information about one or more app/API contracts.

This object is used by the following operations:

  • GET /api/apps/versions/{AppVersionID}/contracts

The properties included in the Contracts object are listed below.

Parameter Type Description
Contract Contract[ ]

Includes information about a contract between an app and an API.

One or more.
