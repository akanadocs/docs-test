APIVersionAutoConnectDetails Object

Contains information about auto-connect details for one or more environments, for an API version.

This object is used by the following operations:

  • PUT /api/apis/versions/{APIVersionID}/autosettings (in the request)

The properties included in the APIVersionAutoConnectDetails object are listed below.

Property Type Description
APIVersionAutoConnectDetail APIVersionAutoConnectDetail[ ]

Contains information about an auto-connect contract for an API version.

One or more.
