DeploymentZoneInfoList Object

Contains information about deployment zones for a specific tenant, in the form of a list.

This object is used by the following operations:

  • GET /api/deploymentzones (with specific headers): in response

The properties included in the DeploymentZoneInfoList object are listed below.

Property Type Description
DeploymentZoneInfo DeploymentZoneInfo[ ]

Contains information about a specific deployment zone on the platform.

One or more.
