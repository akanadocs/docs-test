APIVersionResourceDetails Object

Contains details about operation-specific scope mapping for an API version, if operation-specific scope mapping is in use for the API version.

This object is used by the following operations:

  • PUT /api/apis/versions/{APIVersionID}/resources
  • GET /api/apis/versions/{APIVersionID}/resources
  • Any operations that use the APIVersion object

The properties included in the APIVersionResourceDetails object are listed below.

Property Type Description
Resource Resource Contains information about a resource, such as a scope, defined for a business.
OperationResourceDetails OperationResourceDetails Contains details about operations associated with an API version and the scopes assigned to them, if operation-specific scope mapping is in use for the API version.
