ChangeUserDomainRequest Object

Contains information about a domain change for a user, with comments.

This object is used by the following operations:

  • PUT /api/users/{UserID}/domains/{DomainID}

The properties included in the ChangeUserStatusRequest object are listed below.

Property Type Required Description
UserDomain string Optional The user's domain. For a list of valid values, see User Status.
Comments string Required Comments entered by the user performing the action.
