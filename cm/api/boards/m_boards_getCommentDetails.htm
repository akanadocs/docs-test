GET /api/boards/items/comments/{CommentID}/details

Returns details about the specified Board comment.

Authorization Roles/Permissions: Must be logged in. Must be an authorized user for the resource. For more information about how authorization and visibility work for Board operations, see Boards in the developer portal.
