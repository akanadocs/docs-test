GET /api/contracts/{ContractID}/appdetails

Returns information about the app that's part of a specific app/API contract.

If you have the ContractID, you can use this operation to get more information about the contract.

Authorization Roles/Permissions: To complete this operation successfully, a user must be a member of the applicable team (App team member or API Admin) or a Site Admin.
