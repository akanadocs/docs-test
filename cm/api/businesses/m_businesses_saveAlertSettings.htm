PUT /api/businesses/{BusinessID}/alertsettings

Updates the configurable alert settings for a business on the platform.

Authorization Roles/Permissions: Must be an authorized administrator: Business Admin or Site Admin

Sample Request

The example below shows a request to update the alert settings for the specified business.

Request URL

https://<hostname>/api/businesses/tenantbusiness.acmepaymentscorp/alertsettings

Sample request headers

Accept: application/json
Content-Type: application/json
X-Csrf-Token_{tenant}: <TokenID>

Sample request body

{
  "AlertCommentWorkflowDefKey" : "workflow:definition:comment",
}

Request Parameters

Response
