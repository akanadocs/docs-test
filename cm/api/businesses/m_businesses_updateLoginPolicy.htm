PUT /api/businesses/{BusinessID}/loginpolicy

Updates login policy settings for a business on the platform.

Authorization Roles/Permissions: Must have permission to modify the business information; a Business Admin.

Sample Request

The example below shows a request to update the login policy for the specified business.

Request URL

https://<hostname>/api/businesses/tenantbusiness.acmepaymentscorp/loginpolicy

Sample request headers

Accept: application/json
Content-Type: application/json
X-Csrf-Token_{tenant}: <TokenID>

Sample request body

{
  "MaxSequentialFailureAttemptsAllowed" : 3,
  "AttemptsPeriod" : 1,
  "SuspensionMinutes" : 30,
  "AutoLoginExternalDomain" : "com.soa.feature.enabled",
  "LoginSessionTimeout" : 0
}
