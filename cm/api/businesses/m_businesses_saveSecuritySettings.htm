PUT /api/businesses/{BusinessID}/securitysettings

Saves security-related settings for a business.

Authorization Roles/Permissions: Must be an authorized administrator: Business Admin or Site Admin

Sample Request

The example below shows a request to disable the UserModifyEmail setting.

Request URL

https://<hostname>/api/businesses/tenantbusiness.acmepayments/securitysettings

Sample request headers

Accept: application/json
Content-Type: application/json
X-Csrf-Token_<tenant>: <TokenID>

    "ReadRequestCSRFSupport" : "com.soa.feature.enabled",

    "EncryptChallengeAnswers" : "com.soa.feature.enabled",

    "UserModifyEmail" : "com.soa.feature.disabled"
}

Response
