PUT /api/businesses/{BusinessID}/groupsettings

Updates the configurable group settings for a business on the platform.

Authorization Roles/Permissions: Must be an authorized administrator: Business Admin or Site Admin

URL

Sample Request

The example below shows a request to update the group settings for the specified business. This request disables ratings and reviews for groups.

Request URL

https://<hostname>/api//businesses/tenantbusiness.acmepaymentscorp/groupsettings

Sample request headers

Accept: application/json
Content-Type: application/json; charset=UTF-8
X-Csrf-Token_acmepaymentscorp: <TokenID>

Sample request body: application/json

{
  "GroupSupport" : "com.soa.feature.enabled",
  "PublicGroupSupport" : "com.soa.feature.enabled",
  "PublicGroupBoardSupport" : "com.soa.feature.enabled",
  "PublicGroupRatingsSupport" : "com.soa.feature.disabled",
  "PublicGroupReviewsSupport" : "com.soa.feature.disabled",
  "PrivateGroupSupport" : "com.soa.feature.enabled",
  "PrivateGroupBoardSupport" : "com.soa.feature.enabled",
  "PrivateGroupRatingsSupport" : "com.soa.feature.disabled",
  "PrivateGroupReviewsSupport" : "com.soa.feature.disabled",
  "GroupMemberWorkflowDefKey" : "workflow:definition:application-team-mbr",
  "GroupCommentWorkflowDefKey" : "workflow:definition:comment"
}

Request Parameters

Parameter Parm Type Data Type Required Description
BusinessID Path string Required The unique ID for a specific business on the platform.
BusinessGroupSettings Body BusinessGroupSettings Required Default settings to apply to all groups within the business, unless specified otherwise at a lower level.

Response

If successful, this operation returns HTTP status code 200, with the updated group settings.
