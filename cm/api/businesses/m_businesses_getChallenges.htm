GET /api/businesses/{BusinessID}/challenges

Returns information about the security challenge questions set up for a business.

Authorization Roles/Permissions: Must have read permission for the business information.

Note: The CSRF header is not required for this operation even when CSRF prevention is turned on in the platform.

Response
