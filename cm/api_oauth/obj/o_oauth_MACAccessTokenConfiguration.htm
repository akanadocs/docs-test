OAuth MACAccessTokenConfiguration Object

Contains information about the the OAuth Provider's configuration settings for MAC access token support, if applicable.

This object is used by the following operations:

  • Any operations that use the OAuth Provider object

The properties included in the MACAccessTokenConfiguration object are listed below.

Property Type Description
TokenType string The access token type. In this context, the value is always MAC.
