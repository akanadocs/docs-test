OAuth ResourceOwner Object

Contains information about the ResourceOwner for an OAuth grant.

The properties included in the ResourceOwner object are listed below.

Property Type Description
ResourceOwnerDomain string The domain for the resource owner.
ResourceOwnerID string The unique ID for the resource owner.
ResourceOwnerName string The name of the resource owner.
