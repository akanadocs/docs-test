OAuth GrantResourceScope Object

Contains information about the resources in a grant OAuth scope.

This object is used by the following operations:

  • Any operations that use the Grant object
  • Any operations that use the ExtensibleGrantDocument object

The properties included in the GrantResourceScope object are listed below.

Property Type Description
Resource Resource

Contains information about an OAuth resource (OAuth scope).

One or more.
