OAuth ModifiedGrants Object

Contains information (GrantID) about one or more OAuth grants that have been modified.

The properties included in the ModifiedGrants object are listed below.

Property Type Description
GrantID[ ] string The unique ID for a specific instance of an OAuth grant.
