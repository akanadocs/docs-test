OAuth ResourceHierarchy Object

Contains information about the hierarchy of resources in an OAuth grant.

This object is used by the following operations:

  • Any operations that use the OAuthProvider object

The properties included in the ResourceHierarchy object are listed below.

Property Type Description
Resource[ ] Resource

Contains information about an OAuth resource (OAuth scope).

One or more.
