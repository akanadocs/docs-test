OAuth GrantStatus Object

Contains information about the status of an OAuth grant. For values, see OAuth GrantStatus values.

GrantStatus is an enumerated object. For values, see OAuth GrantStatus.

This object is used by the following operations:

  • Any operations that use the Grant object
  • Any operations that use the ExtensibleGrantDocument object
