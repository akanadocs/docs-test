OpenIdConnectAuzRequestParameters Object

Contains information about the request parameters for an OpenID Connect authorization request.

This object is used by the following operations:

  • Any operations that use the Grant object
  • Any operations that use the ExtensibleGrantDocument object

The properties included in the OpenIdConnectAuzRequestParameters object are listed below.

Property Type Description
Parameter[ ] Parameter

Contains information about an OAuth parameter; a name/value pair.

One or more.
