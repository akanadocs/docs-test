OAuth ChangeStateRequest Object

Contains information about a request to change the status of an OAuth grant.

The properties included in the ChangeStateRequest object are listed below.

Property Type Description
GrantStatus GrantStatus Contains information about the status of an OAuth grant. For values, see OAuth GrantStatus values.
