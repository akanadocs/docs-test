OAuth GrantProperties Object

Contains information about OAuth grant properties.

This object is used by the following operations:

  • PUT /oauth/admin/grants/{GrantID}/properties
  • Any operations that use the GrantRequest object
  • Any operations that use the Grant object

The properties included in the GrantProperties object are listed below.

Property Type Description
GrantProperty GrantProperty[ ]

Contains information about an OAuth grant property.

One or more.
