OAuth ResourceServerGrants Object

Contains information (GrantID) about one or more resource server OAuth grants.

The properties included in the ResourceServerGrants object are listed below.

Property Type Description
ModifiedGrants ModifiedGrants Contains information (GrantID) about one or more OAuth grants that have been modified.
InactiveGrants InactiveGrants Contains information (GrantID) about one or more OAuth grants that are inactive.
DeletedClients[ ] string Optional. Contains information about one or more clients that have been deleted.
