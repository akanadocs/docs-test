Authorization Server Token Admin Service: Overview

Provides a function for revoking an existing OAuth grant.

URL: https://{oauth-provider-url}/oauth/admin/token

Action Endpoint Description
revoke token PUT oauth/admin/token/revoke Revokes an existing OAuth grant.
