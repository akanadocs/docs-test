Welcome to the SOA Software Documentation Repository

The SOA Software Documentation Repository provides documentation for SOA Software's products, including Community Manager, API Gateway (Policy Manager, Network Director, Agents), and Lifecycle Manager, along with a set of documentation covering the product Deployment Platform. The repository is designed to provide quick access to installation, configuration, concept, and usage documentation through use of the search facility. Documentation is accessible in either PDF format (downloadable), or HTML (view / print).
