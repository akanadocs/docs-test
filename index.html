Welcome to the Akana Documentation Repository

The Akana Documentation Repository provides documentation for Akana's products, including Community Manager, API Gateway (Policy Manager, Network Director), Envision, and Lifecycle Manager, along with a set of documentation covering the product Deployment Platform. The repository is designed to provide quick access to installation, configuration, concept, and usage documentation through use of the search facility. Documentation is accessible in either PDF format (downloadable), or HTML (view / print).

For information about new features in the latest release, and updates to earlier releases, see Akana Release Notes.
